# IEnumDebugExtendedPropertyInfo::Reset
Réinitialise une séquence d'énumération.  
  
## Syntaxe  
  
```  
HRESULT Reset (  
void  
);  
```  
  
## Valeur de retour  
 Retourne `HRESULT`valide, en général `S_OK`.
