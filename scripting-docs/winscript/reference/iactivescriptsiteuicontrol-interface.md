# IActiveScriptSiteUIControl, interface
Fournit une méthode pour obtenir le comportement d'un contrôle d'interface utilisateur.  
  
## Méthodes  
  
|Méthode|Description|  
|-------------|-----------------|  
|[IActiveScriptSiteUIControl::GetUIBehavior, méthode](../../winscript/reference/iactivescriptsiteuicontrol-getuibehavior-method.md)|Obtient [SCRIPTUICHANDLING, énumération](../../winscript/reference/scriptuichandling-enumeration.md) qui représente la manière dont un contrôle d'IU doit être exécuté.|
