# SCRIPT_E_PROPAGATE, code d&#39;erreur
Une erreur de script est propagée à l'appelant, qui peut être dans un thread différent.  L'hôte doit passer un code d'erreur à l'appelant.  
  
## Syntaxe  
  
```cpp  
SCRIPT_E_PROPAGATE  0x80020102L  
```
