# IScriptScriptlet::SetEventName
Définit le nom de l'événement associé au scriptlet.  
  
## Syntaxe  
  
```  
HRESULT SetEventName(  
   LPCOLESTR          psz  
);  
```  
  
#### Paramètres  
 `psz`  
 \[in\]  Une mémoire tampon qui contient le nom de l'événement associé à l'objet d' `IScriptScriptlet` .  
  
## Valeur de retour  
 Élément `HRESULT`.  Les valeurs possibles incluent, mais ne sont pas limitées à, celles dans le tableau suivant.  
  
|Valeur|Description|  
|------------|-----------------|  
|`S_OK`|La méthode a réussi.|
