# <a name="ijsenumdebugpropertygetcount-method"></a>IJsEnumDebugProperty::GetCount, méthode
Obtient le nombre de propriétés de l’objet.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
HRESULT GetCount(  
   ULONG *pCount  
);  
```  
  
#### <a name="parameters"></a>Paramètres  
 `pCount`  
 [out] Le nombre de propriétés.  
  
## <a name="return-value"></a>Valeur de retour
