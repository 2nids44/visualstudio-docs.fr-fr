# IDebugPropertyEnumType_All::GetName
Retourne un BSTR contenant le nom d' `EnumType`.  
  
## Syntaxe  
  
```  
HRESULT GetName(  
   BSTR*  pname  
);  
```  
  
#### Paramètres  
 `pname`  
 \[out\]  BSTR contenant le nom d' `EnumType`.  
  
## Valeur de retour  
 Retourne `HRESULT`valide, en général `S_OK`.
