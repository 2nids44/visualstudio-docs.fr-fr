# IRemoteDebugApplicationThreadEx (interface)
Représente un thread d'exécution dans une application particulière.  
  
 Outre les méthodes héritées de `IUnknown`, l'interface `IRemoteDebugApplicationThreadEx` expose les méthodes suivantes.  
  
## Méthodes dans l'ordre Vtable  
  
|Méthode|Description|  
|-------------|-----------------|  
|[IRemoteDebugApplicationThreadEx:EnumGlobalContexts](../../winscript/reference/iremotedebugapplicationthreadex-enumglobalcontexts.md)|Énumère les contextes globaux d'expression pour tous les langages exécution de ce thread.|
