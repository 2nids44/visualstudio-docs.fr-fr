# <a name="iremotedebugapplicationthreadex-interface"></a>IRemoteDebugApplicationThreadEx (interface)
Représente un thread d’exécution dans une application particulière.  
  
 Outre les méthodes héritées de `IUnknown`, le `IRemoteDebugApplicationThreadEx` interface expose les méthodes suivantes.  
  
## <a name="methods-in-vtable-order"></a>Méthodes dans l'ordre Vtable  
  
|Méthode|Description|  
|------------|-----------------|  
|[IRemoteDebugApplicationThreadEx:EnumGlobalContexts](../../winscript/reference/iremotedebugapplicationthreadex-enumglobalcontexts.md)|Énumère les contextes d’expression globale pour toutes les langues en cours d’exécution dans ce thread.|
