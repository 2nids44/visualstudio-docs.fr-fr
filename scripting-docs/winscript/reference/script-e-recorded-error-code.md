# SCRIPT_E_RECORDED, code d&#39;erreur
Une erreur a été passée entre le moteur de script et l'hôte.  Les besoins hôte de passer un code d'erreur à l'appelant.  
  
## Syntaxe  
  
```cpp  
SCRIPT_E_RECORDED   0x86664004L  
```
