# IActiveScriptTraceInfo::StopScriptTracing, m&#233;thode
Suivi de script de points.  
  
## Syntaxe  
  
```  
HRESULT StopScriptTracing(void);   
}  
  
```  
  
#### Paramètres  
 Cette méthode n'a pas de paramètre.  
  
## Valeur de retour  
 Les valeurs de retour possibles de cette méthode sont les suivantes :  
  
1.  S\_OK : succès.  
  
2.  E\_NOTIMPL : non implémenté.
