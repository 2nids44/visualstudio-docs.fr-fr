# IActiveScriptGarbageCollector, interface
Fournit une méthode au garbage collection de début.  Cette interface doit être implémentée par les moteurs de script actif qui souhaitent nettoyer leurs ressources.  
  
## Méthodes  
  
|Méthode|Description|  
|-------------|-----------------|  
|[IActiveScriptGarbageCollector::CollectGarbage](../../winscript/reference/iactivescriptgarbagecollector-collectgarbage.md)|Démarre le garbage collection.|
