# IDebugDocument, interface
Toute interface de base pour les versions debug des documents.  
  
## Remarques  
 L'interface d' `IDebugDocument` expose uniquement les méthodes héritées d' [IDebugDocumentInfo, interface](../../winscript/reference/idebugdocumentinfo-interface.md).  Elle n'expose pas de méthode supplémentaire.
