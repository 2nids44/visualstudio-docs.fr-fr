# IBindEventHandler, interface
Permet à l'appelant de fournir un objet qui gère un gestionnaire d'événements spécifié.  
  
## Méthodes  
 Outre les méthodes héritées de `IUnknown`, l'interface `IBindEventHandler` expose les méthodes suivantes.  
  
|Méthode|Description|  
|-------------|-----------------|  
|[IBindEventHandler::BindHandler](../../winscript/reference/ibindeventhandler-bindhandler.md)|Lie un événement à un objet.|
