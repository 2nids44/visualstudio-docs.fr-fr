# IJsDebugBreakPoint::Delete, m&#233;thode
Supprime le point d'arrêt.  
  
## Syntaxe  
  
```  
HRESULT Delete(void);  
```  
  
## Valeur de retour  
  
## Notes  
 Retourne S\_FALSE s'il est appelé sur un point d'arrêt supprimé.  
  
## Configuration requise  
 **En\-tête :** jscript9diag.h
