# IEnumDebugPropertyInfo, interface
Énumère les structures d' `DebugPropertyInfo` .  
  
## Méthodes dans l'ordre Vtable  
 Le tableau suivant répertorie les méthodes d' `IEnumDebugPropertyInfo`.  
  
|Méthode|Description|  
|-------------|-----------------|  
|[IEnumDebugPropertyInfo::Next](../../winscript/reference/ienumdebugpropertyinfo-next.md)|Récupère un nombre spécifié de structures d' `DebugPropertyInfo` dans une séquence d'énumération.|  
|[IEnumDebugPropertyInfo::Skip](../../winscript/reference/ienumdebugpropertyinfo-skip.md)|Ignore un nombre spécifié de structures d' `DebugPropertyInfo` dans une séquence d'énumération.|  
|[IEnumDebugPropertyInfo::Reset](../../winscript/reference/ienumdebugpropertyinfo-reset.md)|Remet au début la séquence d'énumération.|  
|[IEnumDebugPropertyInfo::Clone](../../winscript/reference/ienumdebugpropertyinfo-clone.md)|Crée un énumérateur qui contient le même état d'énumération que l'énumérateur actuel.|  
|[IEnumDebugPropertyInfo::GetCount](../../winscript/reference/ienumdebugpropertyinfo-getcount.md)|Obtient le nombre de structures d' `DebugPropertyInfo` dans un énumérateur.|  
  
## Configuration requise  
 En\-tête : dbgprop.h
