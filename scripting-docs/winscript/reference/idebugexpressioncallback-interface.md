# IDebugExpressionCallBack, interface
Fournit des événements de mode rapport avec la progression d'une évaluation d' `IDebugExpression` .  
  
## Méthodes  
 Outre les méthodes héritées de `IUnknown`, l'interface `IDebugExpressionCallBack` expose les méthodes suivantes.  
  
|Méthode|Description|  
|-------------|-----------------|  
|[IDebugExpressionCallBack::onComplete](../../winscript/reference/idebugexpressioncallback-oncomplete.md)|Indique que l'évaluation de l'expression est terminée.|
