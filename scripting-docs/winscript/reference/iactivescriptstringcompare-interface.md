# IActiveScriptStringCompare, interface
l'interface d' `IActiveScriptCompare` est utilisée pour définir la méthode de comparaison de chaînes pour le moteur de script.

|Méthode|Description|  
|-------------|-----------------|  
|[IActiveScriptStringCompare::StrComp](../../winscript/reference/iactivescriptstringcompare-strcomp.md)|Définit la méthode de comparaison de chaînes pour le moteur de script.|
