# ICanHandleException, interface
Permet à l'appelant d'un moteur de script pour spécifier les exceptions l'appelant gère.  
  
## Méthodes  
 Outre les méthodes héritées de `IUnknown`, l'interface `ICanHandleException` expose les méthodes suivantes.  
  
|Méthode|Description|  
|-------------|-----------------|  
|[ICanHandleException::CanHandleException](../../winscript/reference/icanhandleexception-canhandleexception.md)|Détermine si l'appel du moteur de script peut gérer une exception spécifiée.|
