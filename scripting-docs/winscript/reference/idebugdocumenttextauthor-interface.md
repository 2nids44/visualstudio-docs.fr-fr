# <a name="idebugdocumenttextauthor-interface"></a>IDebugDocumentTextAuthor, interface
Fourni par les documents texte qui prise en charge la modification.  
  
 Outre les méthodes héritées de `IDebugDocumentText`, le `IDebugDocumentTextAuthor` interface expose les méthodes suivantes.  
  
## <a name="methods-in-vtable-order"></a>Méthodes dans l'ordre Vtable  
  
|Méthode|Description|  
|------------|-----------------|  
|[IDebugDocumentTextAuthor::InsertText](../../winscript/reference/idebugdocumenttextauthor-inserttext.md)|Insère un nouveau texte dans le document.|  
|[IDebugDocumentTextAuthor::RemoveText](../../winscript/reference/idebugdocumenttextauthor-removetext.md)|Supprime le texte du document.|  
|[IDebugDocumentTextAuthor::ReplaceText](../../winscript/reference/idebugdocumenttextauthor-replacetext.md)|Remplace du texte dans le document.|
