# IDebugDocumentTextAuthor, interface
Fourni par les documents texte qui prennent en charge la modification.  
  
 Outre les méthodes héritées de `IDebugDocumentText`, l'interface `IDebugDocumentTextAuthor` expose les méthodes suivantes.  
  
## Méthodes dans l'ordre Vtable  
  
|Méthode|Description|  
|-------------|-----------------|  
|[IDebugDocumentTextAuthor::InsertText](../../winscript/reference/idebugdocumenttextauthor-inserttext.md)|Insère le nouveau texte dans le document.|  
|[IDebugDocumentTextAuthor::RemoveText](../../winscript/reference/idebugdocumenttextauthor-removetext.md)|Supprime le texte du document.|  
|[IDebugDocumentTextAuthor::ReplaceText](../../winscript/reference/idebugdocumenttextauthor-replacetext.md)|Remplace le texte dans le document.|
