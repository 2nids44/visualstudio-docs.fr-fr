# IJsDebugProcess::PerformAsyncBreak, m&#233;thode
Place le moteur de script en mode arrêt, ce qui provoque son arrêt sur l'instruction de script suivante.  
  
## Syntaxe  
  
```  
HRESULT PerformAsyncBreak(  
   DWORD threadId  
);  
```  
  
#### Paramètres  
 `threadId`  
 \[in\] ID de thread.  
  
## Valeur de retour  
  
## Configuration requise
