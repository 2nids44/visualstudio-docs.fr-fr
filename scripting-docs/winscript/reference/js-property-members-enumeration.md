# &#201;num&#233;ration JS_PROPERTY_MEMBERS
Indicateurs pour spécifier le type d'informations à retourner dans une requête pour les membres d'un objet.  
  
## Syntaxe  
  
```  
enum JS_PROPERTY_MEMBERS  
{  
   JS_PROPERTY_MEMBERS_ALL = 0,  
   JS_PROPERTY_MEMBERS_ARGUMENTS = 1  
} JS_PROPERTY_MEMBERS;  
```  
  
## Membres  
  
### Valeurs  
  
|Nom|Description|  
|---------|-----------------|  
|`JS_PROPERTY_MEMBERS_ALL`|Représente une demande d'énumérer tous les membres.|  
|`JS_PROPERTY_MEMBERS_ARGUMENTS`|Représente une demande d'énumérer des arguments uniquement.|
