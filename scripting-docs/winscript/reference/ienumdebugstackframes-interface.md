# IEnumDebugStackFrames, interface
Énumère les frames de pile correspondant à un thread.  
  
## Méthodes  
 Outre les méthodes héritées de `IUnknown`, l'interface `IEnumDebugStackFrames` expose les méthodes suivantes.  
  
|Méthode|Description|  
|-------------|-----------------|  
|[IEnumDebugStackFrames::Next](../../winscript/reference/ienumdebugstackframes-next.md)|Récupère un nombre spécifié de segments de la séquence d'énumération.|  
|[IEnumDebugStackFrames::Skip](../../winscript/reference/ienumdebugstackframes-skip.md)|Ignore un nombre spécifié de segments dans une séquence d'énumération.|  
|[IEnumDebugStackFrames::Reset](../../winscript/reference/ienumdebugstackframes-reset.md)|Réinitialise une séquence d'énumération.|  
|[IEnumDebugStackFrames::Clone](../../winscript/reference/ienumdebugstackframes-clone.md)|Crée un énumérateur qui contient le même état que l'énumérateur en cours.|
