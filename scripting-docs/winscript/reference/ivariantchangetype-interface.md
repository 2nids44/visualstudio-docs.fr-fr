# IVariantChangeType, interface
Permet à l'appelant de convertir une valeur VARIANT d'un type en un autre.  
  
## Méthodes  
 Outre les méthodes héritées de `IUnknown`, l'interface `IVariantChangeType` expose les méthodes suivantes.  
  
|Méthode|Description|  
|-------------|-----------------|  
|[IVariantChangeType::ChangeType](../../winscript/reference/ivariantchangetype-changetype.md)|Prend une valeur variant et crée un nouveau variant avec un type spécifié.|
