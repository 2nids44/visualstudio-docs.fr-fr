# IActiveScriptStats::ResetStats
Réinitialise les statistiques pour ce script.  
  
## Syntaxe  
  
```  
HRESULT ResetStats();  
```  
  
#### Paramètres  
 Cette méthode n'accepte pas de paramètres.  
  
## Valeur de retour  
 La méthode retourne `HRESULT`.  Les valeurs possibles incluent, mais ne sont pas limitées à, celles dans le tableau suivant.  
  
|Valeur|Description|  
|------------|-----------------|  
|`S_OK`|La méthode a réussi.|  
  
## Notes  
 Cette méthode réinitialise les statistiques pour ce script.
