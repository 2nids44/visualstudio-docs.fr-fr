# Nombre attendu
Vous avez tenté d'appeler les méthodes **Number.prototype.toString** ou **Number.prototype.valueOf** sur un objet n'étant pas du type **Number**.  L'objet de ce type d'appel doit être de type **Number**.  
  
### Pour corriger cette erreur  
  
-   Seules les méthodes **Number.prototype.toString** ou **Number.prototype.valueOf** peuvent être appelées sur des objets de type **Number**.
