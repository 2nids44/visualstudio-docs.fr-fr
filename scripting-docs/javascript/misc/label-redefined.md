# &#201;tiquette red&#233;finie
Vous avez créé une nouvelle étiquette, mais lui avez donné le nom d'une étiquette existante.  Les étiquettes peuvent être utilisées pour marquer des blocs de code, mais, dans une portée spécifiée, elles doivent être uniques.  
  
### Pour corriger cette erreur  
  
-   Vérifiez que toutes les étiquettes que vous utilisez dans vos programmes sont uniques dans leurs portées respectives.
