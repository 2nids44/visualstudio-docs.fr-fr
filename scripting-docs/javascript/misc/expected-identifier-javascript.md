# Identificateur attendu (JavaScript)
Vous avez utilisé un élément autre qu'un identificateur dans un contexte qui en exige un.  Un identificateur peut être :  
  
-   une variable,  
  
-   une propriété,  
  
-   un tableau,  
  
-   un nom de fonction.  
  
### Pour corriger cette erreur  
  
-   Modifiez l'expression de sorte qu'un identificateur apparaisse à gauche du signe égal.
