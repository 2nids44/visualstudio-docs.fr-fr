# <a name="expected-39end39"></a>Attendu &#39; @end&#39;
Vous a tenté de créer un bloc de code compilé conditionnellement, mais n’inclut pas le  **@end**  instruction à la fin. [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)]instructions peuvent être compilées de manière conditionnelle en les plaçant dans un  **@if/ @end**  bloc.  
  
### <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Ajouter à le  **@end**  instruction.
