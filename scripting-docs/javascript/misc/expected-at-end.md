# &#39;@end&#39; attendu
Vous avez tenté de créer un bloc de code compilé de façon conditionnelle, mais avez omis l'instruction **@end** à la fin.  Les instructions de [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] peuvent être compilées de façon conditionnelle en les plaçant dans un bloc **@if\/@end**.  
  
### Pour corriger cette erreur  
  
-   Ajoutez l'instruction **@end** correspondante.
