# &#39;(&#39; attendu (JavaScript)
Vous avez essayé de placer une expression entre parenthèses, mais avez omis la parenthèse ouvrante.  Certaines expressions doivent être entourées par une parenthèse ouvrante et une parenthèse fermante.  Notez l'utilisation des parenthèses dans l'exemple suivant.  
  
```javascript  
for (initialize; test; increment) {  
statement;  
}  
```  
  
### Pour corriger cette erreur  
  
-   Ajoutez la parenthèse gauche à l'expression d'évaluation.
