# &#39;)&#39; attendu (JavaScript)
Vous avez tenté de placer une expression entre parenthèses, mais avez omis la parenthèse fermante.  Une expression doit être incluse entre une parenthèse ouvrante et une parenthèse fermante.  Notez l'utilisation des parenthèses dans l'exemple suivant.  
  
```javascript  
for (initialize; test; increment) {  
statement;  
}  
```  
  
### Pour corriger cette erreur  
  
-   Ajoutez la parenthèse droite à l'expression d'évaluation.
