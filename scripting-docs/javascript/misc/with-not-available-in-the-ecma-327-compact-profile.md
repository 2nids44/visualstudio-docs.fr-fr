# &#39;with&#39; non disponible dans le profil compact&#160;ECMA&#160;327
'with' non disponible dans le profil compact ECMA 327\(ErrWithNotInCP  
  
 Vous n'êtes pas autorisé à utiliser le mot clé « with » dans le profil compact.  
  
 Pour plus d'informations sur le mot clé « with », consultez [Instruction with](../../javascript/reference/with-statement-javascript.md).
