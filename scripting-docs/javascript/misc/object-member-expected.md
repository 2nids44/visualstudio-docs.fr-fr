# Membre objet attendu
Vous avez tenté d'effectuer une opération sur le membre de données d'un objet, mais ce membre n'existe pas.  
  
### Pour corriger cette erreur  
  
-   Vérifiez que vous avez correctement tapé le nom du membre de données.
