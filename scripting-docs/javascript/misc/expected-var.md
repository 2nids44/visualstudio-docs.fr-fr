# &#39;var&#39; attendu
« Var » attendu \(ERRnoVar 1034\)\)  
  
 Vous n'avez pas ajouté le mot clé var alors qu'il était requis.  
  
 Pour plus d'informations sur le mot clé var, consultez [Instruction var](../../javascript/reference/var-statement-javascript.md).
