# &#39;=&#39; attendu (JavaScript)
Lors de la création de la variable à utiliser avec des instructions de compilation conditionnelle, vous avez omis le signe égal entre la variable et la valeur que vous voulez lui assigner.  
  
### Pour corriger cette erreur  
  
-   Ajoutez le signe égal.  Par exemple :  
  
    ```javascript  
    @set @myvar1 = 1  
    ```
