# &#39;:&#39; attendu
Vous avez tenté de créer une expression à l'aide de l'opérateur conditionnel ternaire, mais avez omis d'insérer le signe deux\-points entre le deuxième et le troisième opérande.  L'opérateur conditionnel ternaire \(trois opérandes\) requiert le signe deux\-points entre le deuxième \(true\) et troisième \(false\) opérande.  
  
### Pour corriger cette erreur  
  
-   Insérez le signe deux\-points entre le deuxième et le troisième opérande.
