# Erreur de syntaxe dans l&#39;expression r&#233;guli&#232;re (JavaScript)
La structure de votre chaîne de recherche enfreint une ou plusieurs règles grammaticales d'une expression régulière [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)].  
  
### Pour corriger cette erreur  
  
-   Vérifiez que la structure de votre chaîne de recherche d'expression régulière respecte la syntaxe d'expression régulière [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)].
