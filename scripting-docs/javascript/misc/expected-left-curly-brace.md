# &#39;{&#39; attendu
Vous n'avez pas ajouté l'accolade de gauche marquant le début du corps de la fonction.  Le code composant le corps d'une fonction, même s'il ne comprend qu'une seule ligne, doit être délimité par des accolades.  Notez que l'usage des accolades est plus strict dans le corps des fonctions que dans les boucles.  
  
### Pour corriger cette erreur  
  
-   Ajoutez l'accolade de gauche marquant le début du corps de la fonction.
