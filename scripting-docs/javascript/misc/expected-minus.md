# &#39;/&#39; attendu
Lors de la création d'un littéral d'expression régulière, vous n'y avez pas inclus l'une des barres obliques \(\/\).  De la même manière que les littéraux de chaîne se composent de caractères entre guillemets, les littéraux d'expression régulière se composent de caractères entre deux barres obliques \(\/\).  
  
### Pour corriger cette erreur  
  
-   Insérez une barre oblique de terminaison pour marquer la fin de l'expression régulière.
