# Commentaire inachev&#233;
Vous avez commencé la création d'un bloc de commentaires multiligne, mais ne l'avez pas correctement terminé.  Les commentaires à plusieurs lignes commencent par une combinaison « \/\* » et se terminent par la combinaison inverse « \*\/ ».  Voici un exemple :  
  
```javascript  
/* This is a comment  
This is another part of the same comment.*/  
```  
  
### Pour corriger cette erreur  
  
-   Veillez à terminer les commentaires de plusieurs lignes par la combinaison « \*\/ ».
