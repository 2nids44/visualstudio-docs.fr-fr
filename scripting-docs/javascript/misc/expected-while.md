# &#39;while&#39; attendu
Vous avez omis la condition `while` dans une boucle `do...while`.  Une instruction **do** doit avoir un test `while` correspondant à la fin du bloc de code.  
  
### Pour corriger cette erreur  
  
-   Incluez l'instruction de test `while` après l'accolade fermante.
