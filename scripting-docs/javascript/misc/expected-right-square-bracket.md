# &#39;]&#39; attendu
Vous avez fait une référence à un élément de tableau, mais avez omis le crochet droit.  Toute expression qui fait référence à un élément de tableau doit inclure des crochets ouvrant et fermant.  
  
### Pour corriger cette erreur  
  
-   Ajoutez le crochet droit à l'expression qui fait référence à l'élément de tableau.
