# L&#39;utilisation d&#39;un futur mot r&#233;serv&#233; pour un identificateur n&#39;est pas valide
1049 : l'utilisation d'un futur mot réservé pour un identificateur n'est pas valide  
  
 Vous ne pouvez pas utiliser un futur mot réservé pour nommer une variable, une fonction ou tout autre identificateur que vous définissez.  
  
 Pour plus d'informations à propos des futurs mots réservés, consultez [Futurs mots réservés JavaScript](../../javascript/reference/javascript-future-reserved-words.md).
