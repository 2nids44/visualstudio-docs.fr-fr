# <a name="the-number-of-fractional-digits-is-out-of-range"></a>Le nombre de fractions est en dehors de la plage
Vous avez tenté de passer un argument non valide à la fonction **Number.prototype.toExponential**. L’argument de la fonction **toExponential()** doit être comprise entre 0 et 20 (inclus).  
  
### <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Vérifiez l’argument de **toExponential()** n’est pas trop grande ou trop petite.
