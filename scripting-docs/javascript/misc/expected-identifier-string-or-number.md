# Identificateur, cha&#238;ne ou nombre attendu(e)
Vous avez utilisé une syntaxe de littéral incorrecte pour déclarer un littéral d'objet.  Les propriétés d'un littéral d'objet doivent être un identificateur, une chaîne ou un nombre.  Un littéral d'objet \(également appelé « initialiseur d'objet »\) se compose d'une liste de paires propriété\/valeur séparées par des virgules et mises entre parenthèses.  Par exemple :  
  
```javascript  
var point = {x:1.2, y:-3.4};  
```  
  
### Pour corriger cette erreur  
  
-   Vérifiez que vous utilisez la syntaxe littérale appropriée.
