# Math.random, fonction (JavaScript)
Retourne un nombre pseudoaléatoire compris entre 0 et 1.  
  
## Syntaxe  
  
```  
  
Math.random( )  
  
```  
  
## Notes  
 Le nombre pseudo\-aléatoire généré est compris entre 0 \(inclus\) et 1 \(exclu\). Autrement dit, le nombre retourné peut être zéro, mais il sera toujours inférieur à un.  Le générateur de nombres aléatoires prend automatiquement une valeur initiale lors du chargement de [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)].
