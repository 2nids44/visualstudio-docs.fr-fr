# valueOf, m&#233;thode (Set) (JavaScript)
Retourne la valeur primitive de l'objet spécifié.  
  
## Syntaxe  
  
```javascript  
setObj.valueOf()  
```  
  
#### Paramètres  
 `setObj`  
 Requis.  Objet `Set`.  
  
## Valeur de propriété\/valeur de retour  
 Retourne l'instance de l'ensemble.
