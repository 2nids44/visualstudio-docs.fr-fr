# Math.trunc, fonction (JavaScript)
Retourne la partie entière d'un nombre en retirant la partie décimale.  
  
## Syntaxe  
  
```vb  
Math.trunc(number)  
```  
  
## Notes  
 L'argument `number` requis est une expression numérique pour laquelle l'entier tronqué est nécessaire.  
  
 Si `number` a la valeur `NaN`, retourne `NaN`.
