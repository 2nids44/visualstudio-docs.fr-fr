# <a name="mathcos-function-javascript"></a>Math.cos, fonction (JavaScript)
Retourne le cosinus d'un nombre.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
Math.cos(radians)   
```  
  
## <a name="remarks"></a>Remarques  
 Requis `radians` argument est une expression numérique qui contient un angle, mesuré en radians.  
  
 La valeur de retour est le cosinus de l’argument numérique de `radians`.
