# Math.floor, fonction (JavaScript)
Retourne le plus grand entier inférieur ou égal à son argument numérique.  
  
## Syntaxe  
  
```  
  
Math.floor(number)   
```  
  
## Notes  
 L'argument *number* requis est une expression numérique.  
  
 La valeur de retour est un nombre entier égal au plus grand entier inférieur ou égal à son argument numérique.
