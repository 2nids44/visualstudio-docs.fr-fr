# <a name="mathcbrt-function-javascript"></a>Fonction Math.cbrt (JavaScript)
Retourne la racine cubique d'un nombre.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
Math.cbrt(  
number  
)   
```  
  
## <a name="remarks"></a>Remarques  
 Si `number` est `NaN`, le résultat est `NaN`.
