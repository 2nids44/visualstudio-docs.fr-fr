# <a name="multiplication-operator--javascript"></a>Multiplication, opérateur (*) (JavaScript)
Multiplie la valeur de deux expressions.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
result = number1*number2  
```  
  
## <a name="parameters"></a>Paramètres  
 *résultat*  
 Toute variable.  
  
 `number1`  
 Toute expression.  
  
 `number2`  
 Toute expression.
