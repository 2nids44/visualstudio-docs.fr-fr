# Multiplication, op&#233;rateur (*) (JavaScript)
Multiplie la valeur de deux expressions.  
  
## Syntaxe  
  
```  
  
result = number1*number2  
```  
  
## Paramètres  
 *result*  
 Toute variable.  
  
 `number1`  
 Toute expression.  
  
 `number2`  
 Toute expression.
