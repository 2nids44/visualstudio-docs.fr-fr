# toString, m&#233;thode (WeakMap) (JavaScript)
Retourne une représentation sous forme de chaîne d'un objet `WeakMap`.  
  
## Syntaxe  
  
```javascript  
weakmapObj.toString()  
```  
  
#### Paramètres  
 `weakmapObj`  
 Requis.  Objet `WeakMap`.  
  
## Valeur de propriété\/valeur de retour  
 Représentation sous forme de chaîne de `WeakMap`.
