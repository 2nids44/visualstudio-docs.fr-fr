# <a name="nan-constant-javascript"></a>NaN, constante (JavaScript)
Une constante spéciale qui spécifie une expression de thatNot n’est pas un nombre.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
NaN   
```  
  
## <a name="remarks"></a>Remarques  
 Le `NaN` constante (pas un nombre) est un membre de la `Global` de l’objet et devient disponible lorsque le moteur de script est initialisé.
