# NaN, constante (JavaScript)
Constante spéciale qui spécifie que Not an expression n'est pas un nombre.  
  
## Syntaxe  
  
```  
NaN   
```  
  
## Notes  
 La constante `NaN` \(Not a Number, pas un nombre\) est un membre de l'objet `Global` et devient disponible lors de l'initialisation du moteur de script.
