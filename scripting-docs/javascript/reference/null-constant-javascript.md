# <a name="null-constant-javascript"></a>Constante null (JavaScript)
`null`Permet d’indiquer qu’une variable ne fait pas référence à des données valides. Ce n’est pas la même chose que [undefined constante](../../javascript/reference/undefined-constant-javascript.md).
