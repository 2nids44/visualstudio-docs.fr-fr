# <a name="multiplication-assignment-operator--javascript"></a>Multiplication, opérateur d'assignation (*=) (JavaScript)
Multiplie la valeur d'une variable par la valeur d'une expression et assigne le résultat à la variable.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
result *= expression  
```  
  
## <a name="parameters"></a>Paramètres  
 *résultat*  
 Toute variable.  
  
 *expression*  
 Toute expression.  
  
## <a name="remarks"></a>Remarques  
 À l’aide de la  **\* =**  opérateur est exactement identique à :  
  
```JavaScript  
result = result * expression  
```
