# Fonction Math.log2 (JavaScript)
Retourne le logarithme de base 2 d'un nombre.  
  
## Syntaxe  
  
```  
Math.log2(x)   
```  
  
## Valeur de retour  
 Si `x` est `NaN`, cette fonction retourne `NaN`.  Si `x` est inférieur à zéro \(0\), cette fonction retourne `NaN`.
