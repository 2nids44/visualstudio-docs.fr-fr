# <a name="division-operator---javascript"></a>Opérateur de division (-) (JavaScript)
Divise la valeur de deux expressions.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
result = number1 / number2   
```  
  
## <a name="parameters"></a>Paramètres  
 `result`  
 Toute variable numérique.  
  
 `number1`  
 Toute expression numérique.  
  
 `number2`  
 Toute expression numérique.
