# Fonction Math.tanh (JavaScript)
Retourne la tangente hyperbolique d'un nombre.  
  
## Syntaxe  
  
```  
  
Math.tanh(  
x  
)  
  
```  
  
## Notes  
 La valeur de retour est la même que `(exp(x) - exp(-x))/(exp(x) + exp(-x))`.  Si `x` est `NaN`, le résultat est `NaN`.
