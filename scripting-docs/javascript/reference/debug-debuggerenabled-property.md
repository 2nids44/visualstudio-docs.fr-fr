# <a name="debugdebuggerenabled-property"></a>Debug.debuggerEnabled, propriété
Détermine si le débogage est activé pour le contexte de script. Le débogage est activé ou désactivé ou non un débogueur est attaché.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
var dbgEnabled = Debug.debuggerEnabled;  
```
