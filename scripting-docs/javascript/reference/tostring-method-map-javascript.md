# toString, m&#233;thode (Map) (JavaScript)
Retourne une représentation d'un mappage sous forme de chaîne.  
  
## Syntaxe  
  
```javascript  
mapObj.toString()  
```  
  
#### Paramètres  
 `mapObj`  
 Requis.  Objet `Map`.  
  
## Valeur de propriété\/valeur de retour  
 Représentation du mappage sous forme de chaîne.
