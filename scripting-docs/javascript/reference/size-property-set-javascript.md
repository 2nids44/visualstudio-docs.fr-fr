# size, propri&#233;t&#233; (Set) (JavaScript)
Retourne le nombre d'éléments d'un ensemble.  
  
## Syntaxe  
  
```javascript  
sizeVar = setObj.size  
```  
  
#### Paramètres  
 `sizeVar`  
 Requis.  Nombre quelconque.  
  
 `setObj`  
 Requis.  Tout objet `Set`.
