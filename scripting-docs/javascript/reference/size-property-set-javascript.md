# <a name="size-property-set-javascript"></a>size, propriété (Set) (JavaScript)
Retourne le nombre d’éléments dans un jeu.  
  
## <a name="syntax"></a>Syntaxe  
  
```JavaScript  
sizeVar = setObj.size  
```  
  
#### <a name="parameters"></a>Paramètres  
 `sizeVar`  
 Obligatoire. N'importe quel nombre.  
  
 `setObj`  
 Obligatoire. Tout objet `Set`.
