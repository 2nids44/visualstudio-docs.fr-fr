# <a name="mathlog1p-function-javascript"></a>Fonction Math.log1p (JavaScript)
Retourne le logarithme naturel de 1 + un nombre.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
Math.log1p(x)   
```  
  
## <a name="return-value"></a>Valeur de retour  
 Si `x` est `NaN`, cette fonction retourne `NaN`. Si `x` est inférieur à zéro (0), cette fonction retourne `NaN`.
