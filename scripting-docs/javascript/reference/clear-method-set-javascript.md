# clear, m&#233;thode (Set) (JavaScript)
Supprime tous les éléments de l'ensemble.  
  
## Syntaxe  
  
```javascript  
setObj.clear()  
```  
  
#### Paramètres  
 `setObj`  
 Requis.  Ensemble à effacer.
