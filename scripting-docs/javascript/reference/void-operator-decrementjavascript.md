# void, op&#233;rateur (JavaScript)
Empêche une expression de retourner une valeur.  
  
## Syntaxe  
  
```  
void expression   
```  
  
## Notes  
 L'argument `expression` représente toute expression valide de [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)].  
  
 L'opérateur `void` procède à l'évaluation de cette expression et retourne la valeur `undefined`.  Cela s'avère très utile lorsqu'une expression doit être évaluée mais que vous ne souhaitez pas voir les résultats apparaître dans le reste du script.
