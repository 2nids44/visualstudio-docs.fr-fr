# <a name="mathlog10-function-javascript"></a>Fonction Math.log10 (JavaScript)
Retourne le logarithme de base 10 d'un nombre.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
Math.log10(x)   
```  
  
## <a name="return-value"></a>Valeur de retour  
 Si `x` est `NaN`, cette fonction retourne `NaN`. Si `x` est inférieur à zéro (0), cette fonction retourne `NaN`.
