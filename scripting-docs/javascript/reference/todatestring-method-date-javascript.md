# toDateString, m&#233;thode (Date) (JavaScript)
Retourne une date sous forme de valeur de chaîne.  
  
## Syntaxe  
  
```  
  
objDate.toDateString( )  
```  
  
## Notes  
 La référence *objDate* requise est un objet `Date`.  
  
 La méthode `toDateString` retourne, dans un format facile à lire, une valeur de chaîne contenant la date selon le fuseau horaire actuel.
