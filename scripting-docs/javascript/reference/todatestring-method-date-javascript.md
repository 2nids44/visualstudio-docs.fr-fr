# <a name="todatestring-method-date-javascript"></a>toDateString, méthode (Date) (JavaScript)
Retourne une date sous la forme d’une valeur de chaîne.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
objDate.toDateString( )  
```  
  
## <a name="remarks"></a>Remarques  
 Requis *objDate* référence est un `Date` objet.  
  
 Le `toDateString` méthode retourne une valeur de chaîne contenant la date dans le fuseau horaire actuel, dans la pratique, format facile à lire.
