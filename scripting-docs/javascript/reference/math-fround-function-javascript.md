# <a name="mathfround-function-javascript"></a>Fonction Math.fround (JavaScript)
Retourne le format en virgule flottante simple précision le plus proche d'un nombre.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
Math.fround(  
number  
)   
```  
  
## <a name="remarks"></a>Remarques  
 Si `number` est `NaN`, le résultat est `NaN`.
