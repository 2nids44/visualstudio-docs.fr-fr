# clear, m&#233;thode (Map) (JavaScript)
Supprime tous les éléments dans un mappage.  
  
## Syntaxe  
  
```javascript  
mapObj.clear()  
```  
  
#### Paramètres  
 `mapObj`  
 Requis.  Mappage à effacer.
