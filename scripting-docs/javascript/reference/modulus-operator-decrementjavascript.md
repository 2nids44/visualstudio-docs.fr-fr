# <a name="modulus-operator--javascript"></a>Modulus, opérateur (JavaScript)
Divise la valeur d’une expression numérique par la valeur d’une autre expression numérique et génère le reste.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
result = expression1 % expression2  
```  
  
## <a name="arguments"></a>Arguments  
 `result`  
 Toute variable.  
  
 `expression1`  
 Toute expression numérique.  
  
 `expression2`  
 Toute expression numérique.
