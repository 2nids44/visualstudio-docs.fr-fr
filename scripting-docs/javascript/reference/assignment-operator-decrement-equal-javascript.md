# Op&#233;rateur d&#39;assignation (=) (JavaScript)
Assigne une valeur à une variable.  
  
## Syntaxe  
  
```  
  
result = expression  
```  
  
## Paramètres  
 `result`  
 Toute variable.  
  
 `expression`  
 Toute expression numérique.  
  
## Notes  
 L'opérateur \= se comporte comme les autres opérateurs, donc les expressions qui le contiennent ont une valeur.  Cela signifie que vous pouvez chaîner les opérateurs d'assignation comme suit : `j = k = l = 0`.  Dans ce cas, `j`, `k` et `l` sont égaux à zéro.
