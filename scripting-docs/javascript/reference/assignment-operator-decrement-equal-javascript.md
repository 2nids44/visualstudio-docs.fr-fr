# <a name="assignment-operator--javascript"></a>Opérateur d'assignation (=) (JavaScript)
Assigne une valeur à une variable.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
result = expression  
```  
  
## <a name="parameters"></a>Paramètres  
 `result`  
 Toute variable.  
  
 `expression`  
 Toute expression numérique.  
  
## <a name="remarks"></a>Remarques  
 Le = opérateur se comporte comme les autres opérateurs, donc les expressions qui contiennent une valeur. Cela signifie que vous pouvez chaîner les opérateurs d’assignation comme suit : `j = k = l = 0`. Dans ce cas `j`, `k`, et `l` égale à zéro.
