# Fonction Math.asinh (JavaScript)
Retourne le sinus hyperbolique inverse d'un nombre.  
  
## Syntaxe  
  
```  
  
Math.asinh(x)   
```  
  
## Notes  
 Si `x` est `NaN`, le résultat est `NaN`.
