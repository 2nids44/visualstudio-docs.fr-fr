# Math.asin, fonction (JavaScript)
Retourne l'arc sinus d'un nombre.  
  
## Syntaxe  
  
```  
  
Math.asin(number)   
```  
  
## Notes  
 L'argument `number` requis est une expression numérique pour laquelle l'arc sinus est nécessaire.  
  
 La valeur de retour est l'arc sinus de l'argument `number`, en radians.
