# Fonction Math.sinh (JavaScript)
Retourne le sinus hyperbolique inverse d'un nombre.  
  
## Syntaxe  
  
```  
  
Math.sinh(x)   
```  
  
## Notes  
 La valeur de retour est la même que `(exp(x) - exp(-x))/2`.  Si x est `NaN`, le résultat est `NaN`.
