# resolvedOptions, m&#233;thode (Intl.Collator)
Retourne un objet contenant des propriétés et des valeurs de l'objet assembleur.  
  
## Syntaxe  
  
```  
collatorObj.resolvedOptions()  
```  
  
#### Paramètres  
 `collatorObj`  
 Objet `Collator` à partir duquel récupérer les informations.  
  
## Notes  
 Les propriétés de l'objet retourné correspondent aux propriétés calculées de l'objet `Collator`.
