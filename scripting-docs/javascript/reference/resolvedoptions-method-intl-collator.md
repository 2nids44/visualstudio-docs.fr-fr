# <a name="resolvedoptions-method-intlcollator"></a>resolvedOptions, méthode (Intl.Collator)
Retourne un objet qui contient les propriétés et les valeurs de l’objet d’assembleur.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
collatorObj.resolvedOptions()  
```  
  
#### <a name="parameters"></a>Paramètres  
 `collatorObj`  
 Le `Collator` pour récupérer les informations d’objet.  
  
## <a name="remarks"></a>Remarques  
 Les propriétés de l’objet retourné correspondent aux propriétés calculées de la `Collator` objet.
