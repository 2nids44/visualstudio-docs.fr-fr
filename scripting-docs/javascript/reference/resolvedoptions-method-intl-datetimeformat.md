# resolvedOptions, m&#233;thode (Intl.DateTimeFormat)
Retourne un objet qui contient des propriétés et des valeurs de l'objet formateur de date\/heure.  
  
## Syntaxe  
  
```  
dateTimeFormatObj.resolvedOptions()  
```  
  
#### Paramètres  
 `dateTimeFormatObj`  
 Objet `DateTimeFormat` à partir duquel récupérer les informations.  
  
## Notes  
 Les propriétés de l'objet retourné correspondent aux propriétés calculées de l'objet `DateTimeFormat`.
