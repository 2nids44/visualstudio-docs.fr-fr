# Math.sin, fonction (JavaScript)
Retourne le sinus d'un nombre.  
  
## Syntaxe  
  
```  
  
Math.sin(radians)   
```  
  
## Notes  
 L'argument `radians` est une expression numérique qui contient un angle mesuré en radians.  
  
 La valeur de retour est le sinus de l'argument numérique.
