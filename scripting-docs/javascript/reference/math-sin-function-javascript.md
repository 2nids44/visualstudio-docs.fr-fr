# <a name="mathsin-function-javascript"></a>Math.sin, fonction (JavaScript)
Retourne le sinus d'un nombre.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
Math.sin(radians)   
```  
  
## <a name="remarks"></a>Remarques  
 Le `radians` argument est une expression numérique qui contient un angle, mesuré en radians.  
  
 La valeur de retour est le sinus de l’argument numérique.
