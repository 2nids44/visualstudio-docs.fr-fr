# valueOf, m&#233;thode (Map) (JavaScript)
Retourne la valeur primitive de l'objet spécifié.  
  
## Syntaxe  
  
```javascript  
mapObj.valueOf()  
```  
  
#### Paramètres  
 `mapObj`  
 Requis.  Objet `Map`.  
  
## Valeur de propriété\/valeur de retour  
 Retourne l'instance du mappage.
