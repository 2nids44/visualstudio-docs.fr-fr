# <a name="resolvedoptions-method-intlnumberformat"></a>resolvedOptions, méthode (Intl.NumberFormat)
Retourne un objet qui contient les propriétés et valeurs de la `NumberFormat` objet formateur.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
numberFormatObj.resolvedOptions()  
```  
  
#### <a name="parameters"></a>Paramètres  
 `numberFormatObj`  
 Le `NumberFormat` pour récupérer les informations d’objet.  
  
## <a name="remarks"></a>Remarques  
 Les propriétés de l’objet retourné correspondent aux propriétés calculées de la `NumberFormat` objet.
