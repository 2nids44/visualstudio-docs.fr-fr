# resolvedOptions, m&#233;thode (Intl.NumberFormat)
Retourne un objet qui contient des propriétés et des valeurs de l'`NumberFormat`objet formateur de nombre.  
  
## Syntaxe  
  
```  
numberFormatObj.resolvedOptions()  
```  
  
#### Paramètres  
 `numberFormatObj`  
 Objet `NumberFormat` à partir duquel récupérer les informations.  
  
## Notes  
 Les propriétés de l'objet retourné correspondent aux propriétés calculées de l'objet `NumberFormat`.
