# <a name="mathpow-function-javascript"></a>Math.pow, fonction (JavaScript)
Retourne la valeur d’une expression de base nécessaire à une puissance spécifiée.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
Math.pow(  
base, exponent)   
```  
  
## <a name="parameters"></a>Paramètres  
 *base*  
 Obligatoire. La valeur de base de l’expression.  
  
 *exposant*  
 Obligatoire. La valeur d’exposant de l’expression.  
  
## <a name="example"></a>Exemple  
 Dans l’exemple suivant, une expression numérique égale à baseexponent retourne 1000.  
  
```JavaScript  
Math.pow(10,3);  
```
