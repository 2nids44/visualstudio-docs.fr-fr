# Math.pow, fonction (JavaScript)
Retourne la valeur d'une expression de base élevée à une puissance spécifiée.  
  
## Syntaxe  
  
```  
  
Math.pow(  
base, exponent)   
```  
  
## Paramètres  
 *base*  
 Obligatoire.  Valeur de base de l'expression.  
  
 *exponent*  
 Obligatoire.  Valeur de l'exposant de l'expression.  
  
## Exemple  
 Dans l'exemple suivant, une expression numérique égale à baseexponent retourne 1000.  
  
```javascript  
Math.pow(10,3);  
```
