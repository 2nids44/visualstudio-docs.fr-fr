# <a name="infinity-constant-javascript"></a>Constante infinie (JavaScript)
Un nombre qui est supérieure à la plus grande nombre à virgule flottante. Retourne une valeur initiale de `Number.POSITIVE_INFINITY`. L’infini négatif (-Infinity) est inférieure à flottante plus petit nombre à virgule.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
Infinity   
```  
  
## <a name="remarks"></a>Remarques  
 Le `Infinity` constant est un membre de la `Global` de l’objet et devient disponible lorsque le moteur de script est initialisé.
