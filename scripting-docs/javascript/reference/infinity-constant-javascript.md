# Constante infinie (JavaScript)
Nombre qui est supérieur au plus grand nombre à virgule flottante.  Retourne une valeur initiale `Number.POSITIVE_INFINITY`.  L'infini négatif \(\-Infinity\) est plus petit que le plus petit nombre à virgule flottante.  
  
## Syntaxe  
  
```  
Infinity   
```  
  
## Notes  
 La constante `Infinity` est un membre de l'objet `Global` et devient disponible lors de l'initialisation du moteur de script.
