# clear, m&#233;thode (WeakMap) (JavaScript)
Supprime tous les éléments d'un objet `WeakMap`.  
  
## Syntaxe  
  
```javascript  
weakmapObj.clear()  
```  
  
#### Paramètres  
 `weakmapObj`  
 Requis.  Objet `WeakMap` à effacer.
