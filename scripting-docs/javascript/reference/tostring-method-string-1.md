# <a name="tostring-method-string-1"></a>ToString, méthode (String) 1
Retourne une représentation sous forme de chaîne d’une chaîne.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
string.toString()  
```  
  
## <a name="parameters"></a>Paramètres  
 `string`  
 Obligatoire. Le tableau pour représenter sous forme de chaîne.  
  
## <a name="return-value"></a>Valeur de retour  
 La représentation sous forme de chaîne de la chaîne.  
  
## <a name="example"></a>Exemple  
 L’exemple suivant illustre l’utilisation de la **toString** méthode avec une chaîne.  
  
```JavaScript  
var string = "this is a test";  
var strStr = string.toString();  
document.write(strStr);  
  
// Output: this is a test  
  
```
