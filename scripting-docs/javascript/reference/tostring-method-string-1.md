# toString, m&#233;thode (String)
Retourne une représentation sous forme de chaîne d'une chaîne.  
  
## Syntaxe  
  
```  
  
string.toString()  
```  
  
## Paramètres  
 `string`  
 Obligatoire.  Tableau à représenter en tant que chaîne.  
  
## Valeur de retour  
 Représentation sous forme de chaîne de la chaîne.  
  
## Exemple  
 L'exemple ci\-dessous illustre l'utilisation de la méthode **toString** avec une chaîne.  
  
```javascript  
var string = "this is a test";  
var strStr = string.toString();  
document.write(strStr);  
  
// Output: this is a test  
  
```
