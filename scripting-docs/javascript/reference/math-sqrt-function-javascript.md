# Math.sqrt, fonction (JavaScript)
Retourne la racine carrée d'un nombre.  
  
## Syntaxe  
  
```  
  
Math.sqrt(  
number  
)   
```  
  
## Notes  
 L'argument `number` requis est une expression numérique.  
  
 Si l'argument `number` est négatif, la valeur de retour est `NaN`.
