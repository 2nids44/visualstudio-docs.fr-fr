# valueOf, m&#233;thode (WeakMap) (JavaScript)
Retourne la valeur primitive de l'objet spécifié.  
  
## Syntaxe  
  
```javascript  
weakmapObj.valueOf()  
```  
  
#### Paramètres  
 `weakmapObj`  
 Requis.  Objet `WeakMap`.  
  
## Valeur de propriété\/valeur de retour  
 Retourne l'instance `WeakMap`.
