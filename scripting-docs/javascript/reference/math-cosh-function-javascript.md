# <a name="mathcosh-function-javascript"></a>Fonction Math.cosh (JavaScript)
Retourne le cosinus hyperbolique d'un nombre.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
Math.cosh(x)   
```  
  
## <a name="remarks"></a>Remarques  
 La valeur de retour est la même que `(exp(x) + exp(-x))/2`. Si `x` est `NaN`, le résultat est `NaN`.
