# toString, m&#233;thode (Set) (JavaScript)
Retourne une représentation sous forme de chaîne d'un ensemble.  
  
## Syntaxe  
  
```javascript  
setObj.toString()  
```  
  
#### Paramètres  
 `setObj`  
 Requis.  Objet `Set`.  
  
## Valeur de propriété\/valeur de retour  
 Représentation sous forme de chaîne du jeu.
