# Math.tan, fonction (JavaScript)
Retourne la tangente d'un nombre.  
  
## Syntaxe  
  
```  
  
Math.tan(  
radians  
)  
  
```  
  
## Notes  
 L'argument `radians` requis est une expression numérique qui contient un angle mesuré en radians.  
  
 La valeur de retour est la tangente de l'argument numérique de l'élément `radians`.
