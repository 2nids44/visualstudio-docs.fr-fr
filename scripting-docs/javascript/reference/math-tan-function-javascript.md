# <a name="mathtan-function-javascript"></a>Math.tan, fonction (JavaScript)
Retourne la tangente d'un nombre.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
Math.tan(  
radians  
)  
  
```  
  
## <a name="remarks"></a>Remarques  
 Requis `radians` argument est une expression numérique qui contient un angle, mesuré en radians.  
  
 La valeur de retour est la tangente de l’argument numérique de `radians`.
