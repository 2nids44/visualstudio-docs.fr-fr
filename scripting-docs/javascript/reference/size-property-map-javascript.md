# size, propri&#233;t&#233; (Map) (JavaScript)
Retourne le nombre d'éléments d'un mappage.  
  
## Syntaxe  
  
```javascript  
sizeVar = mapObj.size  
```  
  
#### Paramètres  
 `sizeVar`  
 Requis.  Nombre quelconque.  
  
 `mapObj`  
 Requis.  Tout objet `Map`.
