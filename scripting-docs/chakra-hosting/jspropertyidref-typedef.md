# JsPropertyIdRef, typedef
Un identificateur de propriété.  
  
## Syntaxe  
  
```  
typedef JsRef JsPropertyIdRef;  
```  
  
## Notes  
 Les identificateurs de propriété sont utilisés à la place de chaînes pour référencer des propriétés d'objets JavaScript.  
  
## Configuration requise  
 **En\-tête :** jsrt.h
