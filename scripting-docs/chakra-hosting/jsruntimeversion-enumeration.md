# <a name="jsruntimeversion-enumeration"></a>JsRuntimeVersion, énumération
Version du runtime.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
enum JsRuntimeVersion;  
```  
  
## <a name="members"></a>Membres  
  
### <a name="values"></a>Valeurs  
  
|Nom|Description|  
|----------|-----------------|  
|`JsRuntimeVersion10`|Crée le runtime avec la version IE10.|  
|`JsRuntimeVersion11`|Crée le runtime avec la version IE11.|  
|`JsRuntimeVersionEdge`|Crée le runtime avec la version la plus récente présente sur l'ordinateur au moment de l'exécution.|

 **En-tête :** jsrt.h
