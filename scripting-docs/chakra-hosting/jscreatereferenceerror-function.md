# <a name="jscreatereferenceerror-function"></a>JsCreateReferenceError, fonction
Crée un objet d'erreur JavaScript ReferenceError  
  
## <a name="syntax"></a>Syntaxe  
  
```  
STDAPI_(JsErrorCode) JsCreateReferenceError(  
   _In_ JsValueRef message,  
   _Out_ JsValueRef *error  
);  
```  
  
#### <a name="parameters"></a>Paramètres  
 `message`  
 Message de l'objet d'erreur.  
  
 `error`  
 Nouvel objet d'erreur.  
  
## <a name="return-value"></a>Valeur de retour  
 Le code `JsNoError` si l'opération a réussi, sinon un code d'échec.  
  
## <a name="remarks"></a>Remarques  
 Nécessite un contexte de script actif.  
  
## <a name="requirements"></a>Spécifications  
 **En-tête :** jsrt.h
