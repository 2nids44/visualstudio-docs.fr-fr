# JsValueRef, typedef
Une référence à une valeur JavaScript.  
  
## Syntaxe  
  
```  
typedef JsRef JsValueRef;  
```  
  
## Notes  
 Une valeur JavaScript est un des types de valeurs suivants : Non défini, Null, Booléen, Chaîne, Nombre ou Objet.  
  
## Configuration requise  
 **En\-tête :** jsrt.h
