# JsGetContextOfObject, fonction
Obtient le contexte de script auquel l’objet appartient.  
  
## Syntaxe  
  
```  
STDAPI_(JsErrorCode) JsGetContextOfObject(  
  _In_ JsValueRef object,  
  _Out_ JsContextRef *context  
);  
```  
  
#### Paramètres  
 `object`  
 Objet auprès duquel le contexte doit être obtenu.  
  
 `context`  
 Contexte auquel l’objet appartient.  
  
## Valeur de retour  
 Le code `JsNoError` si l'opération a réussi, sinon un code d'échec.  
  
## Notes  
 Nécessite un contexte de script actif.  
  
## Configuration requise  
 **En\-tête :** jsrt.h
