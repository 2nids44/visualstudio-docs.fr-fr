# <a name="activating-the-power-bi-pro-benefit-in-visual-studio-subscriptions"></a>Activation de l’avantage Power BI Pro dans les abonnements Visual Studio

Développez, testez et déployez des compléments pour Office et SharePoint.  Inclut un site de développement SharePoint Online pour un locataire. 

Pour utiliser Power BI Pro, vous devez créer un compte ou vous connecter à l’aide d’un compte existant. 
1.  Cliquez sur le lien **Obtenir un code** en bas de la vignette de l’avantage.   Vous recevez une notification confirmant l’envoi du code.  Cliquez sur **Activer**. 

    ![Vignette de l’avantage Power BI Pro](_img\vs-pbi\vs-pbi-tile.png)  


2. Vous avez peut-être déjà un compte si vous avez activé des avantages d’autres abonnements, tels que l’abonnement Office 365 Developer ou Enterprise Mobility + Security.  Si effectivement vous avez un compte, vous avez le choix d’ajouter votre abonnement Power BI Pro à ce compte ou de vous inscrire pour créer un autre compte.  Nous vous recommandons vivement d’ajouter l’abonnement à votre compte existant pour pouvoir accéder à vos divers avantages à l’aide d’un identifiant utilisateur unique.  

    ![Avantage Power BI Pro, Compte existant](_img\vs-pbi\vs-pbi-existing-account.png) 

3.  Pour créer un compte, entrez vos coordonnées, puis cliquez sur **Suivant**.

    ![Avantage Power BI Pro, Créer un compte](_img\vs-pbi\vs-pbi-create-account-cropped.png) 


4.  Créez ensuite votre identifiant utilisateur et votre mot de passe.  Si vous avez déjà un compte et que vous en créez un autre, vous devez spécifier un nom de société différent de celui que vous avez utilisé pour le compte existant.  Cliquez sur **Créer mon compte**.

    ![Avantage Power BI Pro, Créer un identifiant utilisateur](_img\vs-pbi\vs-pbi-create-user-id-cropped.png) 


5.  Pour prouver que le compte est bien créé par une personne réelle, vous êtes invité à entrer un code de confirmation.  Entrez un numéro de téléphone et choisissez si vous souhaitez recevoir votre code via un appel téléphonique ou un SMS.  Cliquez sur **Envoyer un SMS** ou **M’appeler**.

    ![Avantage Power BI Pro, Vous n’êtes pas un robot](_img\vs-pbi\vs-pbi-robot1-cropped.png) 


6.  Entrez le code reçu, puis cliquez sur **Suivant**. 

    ![Avantage Power BI Pro, Vous n’êtes pas un robot 2](_img\vs-pbi\vs-pbi-robot2-cropped.png) 

7.  Entrez vos coordonnées, puis cliquez sur **Suivant**. 

    ![Avantage Power BI Pro, Coordonnées](_img\vs-pbi\vs-pbi-contact-cropped.png)


8.  La page suivante confirme que votre abonnement Power BI Pro Developer (MSDN) coûte 0,00 €.  Cliquez sur **Suivant**.

9.   Pour finir, vous devez entrer certaines informations de paiement.  Ne vous inquiétez pas. Aucun montant ne sera prélevé sur votre compte.  Notez que le prix est toujours de 0,00 €.  Vous pouvez choisir une carte de paiement, un compte bancaire ou une facture comme mode de paiement.  Si vous choisissez **Facture**, vous pouvez être invité à indiquer les raisons pour lesquelles vous préférez ce mode de paiement.  Entrez les informations demandées pour le mode de paiement choisi, puis cliquez sur **Passer commande**.   
    ![Avantage Power BI Pro, Coordonnées](_img\vs-pbi\vs-pbi-payment-blurred-cropped.png)

10. La page suivante confirme que vous êtes prêt à commencer à utiliser Power BI Pro.  Cette page contient un lien pour accéder à la page de connexion.  Nous vous recommandons d’enregistrer le lien vers la page de connexion, ainsi que votre identifiant utilisateur.  (Vous recevez aussi un e-mail de bienvenue à l’adresse que vous avez fournie au moment de la création de votre compte.)  Vous pouvez également voir s’afficher une fenêtre Assistant Office 365 automatisée, qui vous permet de poser des questions au besoin.  Cliquez sur **Démarrer** pour continuer.

    ![Avantage Power BI Pro, Tout est prêt](_img\vs-pbi\vs-pbi-all-set-cropped.png) 


11. Votre compte Power BI Pro est créé. Vous avez maintenant accès au Centre d’administration Office 365, où vous pouvez gérer les utilisateurs et les domaines, installer des logiciels Office et obtenir plus d’informations sur Office 365.  C'est tout !  Vous êtes prêt à commencer.
