# <a name="visual-studio-subscriptions"></a>Abonnements Visual Studio
##  <a name="developer-tools-cloud-services-software-support-and-training"></a>Outils de développement, services cloud, logiciels, support et formation

Les abonnements Visual Studio sont l’ensemble le plus complet de ressources vous permettant de créer, déployer et gérer de formidables applications sur vos plateformes et appareils préférés, notamment Windows, iOS, Android et Linux.

# <a name="welcome-to-subscriptions"></a>Bienvenue dans les abonnements !

Que vous soyez un abonné ou que vous gériez les abonnements de votre entreprise, vous trouverez des informations qui vous aideront à tirer le meilleur parti des abonnements Visual Studio.
