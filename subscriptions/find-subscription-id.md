# <a name="locating-your-subscription-id-for-visual-studio-subscriptions"></a>Rechercher des ID d’abonnement Visual Studio

## <a name="what-is-a-subscription-id"></a>Qu’est-ce qu’un ID d’abonnement ?
L’ID d’abonnement est un identificateur global unique (GUID) de 32 chiffres associé à un abonnement ou à un programme d’abonné.  Dans certains cas, l’équipe de support peut demander à un abonné à fournir son ID d’abonné pour réduire le risque de problèmes liés à un abonnement spécifique. 

## <a name="how-to-find-your-subscription-id"></a>Rechercher l’ID d’abonnement
1. Accédez à [https://my.visualstudio.com/subscriptions](https://my.visualstudio.com/subscriptions?wt.mc_id=o~msft~docs).
2. Connectez-vous à votre abonnement.  (Pour obtenir de l’aide, [connectez-vous](/visualstudio/subscriptions/signing-in).)
3. Une fois connecté, vous verrez une liste des abonnements associés à l’adresse e-mail que vous avez utilisée pour vous connecter.  La page affiche les éléments suivants :
    - Type de programme ou d’abonnement
    - ID de l'abonnement 
    - Type (vente au détail, NFR, etc.)
    - Période de validité  (date d’expiration)
    - Liens pour renouveler l’abonnement ou le programme, pour obtenir des informations de support et, dans le cas de Visual Studio Dev Essentials, pour quitter le programme
