# <a name="activation"></a>Activation

Visual Studio pour Mac offre trois niveaux d’abonnement différents, tous accessibles à partir d’un seul téléchargement :

* Visual Studio Enterprise pour Mac
* Visual Studio Professional pour Mac
* Visual Studio Community pour Mac

Pour commencer à utiliser un abonnement, connectez-vous à Visual Studio pour Mac en suivant les étapes décrites dans le guide [Connexion à Visual Studio pour Mac](~/signing-in.md). Cette action active automatiquement votre licence et l’affiche dans la boîte de dialogue des comptes :

![Boîte de dialogue Afficher la licence utilisateur](media/user-accounts-login.png)

## <a name="update-expired-licenses"></a>Mettre à jour des licences ayant expiré 

Si votre abonnement a expiré, vous avez deux options :

1. Renouveler votre abonnement. Vous pouvez renouveler votre abonnement en accédant à [https://www.visualstudio.com/vs/pricing/](https://www.visualstudio.com/vs/pricing/).

2. Si vous avez un autre compte avec un abonnement actif, vous pouvez l’ajouter à Visual Studio pour Mac comme indiqué dans la rubrique [Ajout de plusieurs comptes d’utilisateur](~/signing-in.md). Visual Studio pour Mac détecte la meilleure licence disponible dans tous les comptes que vous avez ajoutés. 

## <a name="offline-activation"></a>Activation hors connexion

> [!NOTE]
> Actuellement, il n’existe pas d’activation hors connexion pour Visual Studio pour Mac. Pour activer votre abonnement, vous devez être connecté à Internet.

Pour activer votre abonnement Visual Studio 2017 pour Mac Professional ou Enterprise, vous devez être connecté à Internet.  

**Remarque**: Actuellement, il n’existe pas d’activation hors connexion pour Visual Studio Professional pour Mac ni pour Visual Studio Enterprise pour Mac. Vous pouvez cependant utiliser Visual Studio 2017 pour Mac dans des environnements sans connexion Internet, conformément à la licence.

Si vous avez acheté Visual Studio Enterprise 2017 pour Mac pour des machines sans connexion Internet, vous pouvez continuer à utiliser Visual Studio pour Mac sans activation, mais Xamarin Inspector, Xamarin Profiler et les fonctionnalités des assemblys incorporés ne seront pas disponibles.
