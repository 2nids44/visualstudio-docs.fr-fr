# <a name="editor-themes"></a>Thèmes de l’éditeur
Les thèmes de l’éditeur peuvent être changés pour un des dix jeux de couleurs fournis par défaut dans Visual Studio pour Mac via les options de menu **Visual Studio > Préférences... > Éditeur de texte > Thèmes de couleurs** :

 ![Sélection du thème de couleurs](media/source-editor-image17.png)

## <a name="adding-new-themes"></a>Ajout de nouveaux thèmes

Vous pouvez ajouter de nouveaux thèmes de couleurs, à condition qu’ils soient dans un des formats pris en charge. Ces équivalents sont :

* Visual Studio (`.vssettings`)
* Xamarin Studio (`.json`)
* TextMate (`.tmTheme`)

Il existe de nombreuses ressources fournies en ligne, qui vous permettent de créer votre thème et de produire un fichier dans le format pris en charge. Il existe également de nombreuses galeries en ligne qui offrent un accès gratuit aux thèmes Visual Studio et TextMate.

Une fois que vous avez trouvé ou créé un nouveau thème, et que vous l’avez téléchargé sur votre machine locale, ajoutez-le en procédant comme suit :

1. Sélectionnez le bouton **ajouter** dans la boîte de dialogue Options des thèmes de couleurs :   
    ![Ajout d’un thème de couleurs](media/source-editor-image20.png)
2. Recherchez le thème sur votre machine locale et cliquez sur **Ouvrir**.
3. Ceci ajoute le thème dans le bas de la liste des thèmes préinstallés. Pour en afficher un aperçu, sélectionnez-le. Ceci met à jour votre éditeur en arrière-plan : ![Aperçu du thème de couleurs](media/source-editor-image21.png)
4. Pour l’appliquer, cliquez sur le bouton **OK**.
