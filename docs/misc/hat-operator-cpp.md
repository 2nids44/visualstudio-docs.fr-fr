# ^, op&#233;rateur (C++)
L'opérateur `^` a deux significations dans Visual C\+\+, selon son contexte.  Pour plus d'informations, consultez :  
  
-   [Opérateur de bits OR exclusif : ^](../Topic/Bitwise%20Exclusive%20OR%20Operator:%20%5E.md), pour une utilisation dans toutes les versions C\+\+.  
  
-   [Handle sur l'opérateur Object \(^\)](/visual-cpp/windows/handle-to-object-operator-hat-cpp-component-extensions), pour une utilisation dans [!INCLUDE[cppwrt_short](../misc/includes/cppwrt_short_md.md)] et [!INCLUDE[cppcli](../misc/includes/cppcli_md.md)].
