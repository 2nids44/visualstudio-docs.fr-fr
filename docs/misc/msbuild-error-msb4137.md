# Erreur MSBuild MSB4137
**MSB4137: Valeur non valide spécifiée dans le fichier de configuration à" &lt ; x &gt ;."  Le nom de la propriété ou le nom de la version des outils est une chaîne vide.**  
  
 Une définition d'ensemble d'outils contient des noms ou valeurs de propriété non valides.  
  
### Pour corriger cette erreur  
  
-   Assurez\-vous que votre définition d'ensemble d'outils ne contient pas de noms ou valeurs de propriété vides ou non valides.
