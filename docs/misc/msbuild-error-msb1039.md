# Erreur MSBuild MSB1039
**MSB1039: Spécifiez la version de l'ensemble d'outils.**  
  
 Aucune valeur n'a été spécifiée lors de l'utilisation du commutateur **\/ToolsVersion**.  
  
### Pour corriger cette erreur  
  
-   Lorsque vous utilisez le commutateur **\/ToolsVersion**, assurez\-vous que vous spécifiez une valeur valide non vide pour `ToolsVersion`.
