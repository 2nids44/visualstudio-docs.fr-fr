# Erreur MSBuild MSB4138
**MSB4138: Des données non\-chaînes ont été spécifiées à l'emplacement de Registre "' \<clé\>'".**  
  
 L'emplacement de Registre spécifié dans une configuration d'ensemble d'outils personnalisée n'est pas valide.  
  
### Pour corriger cette erreur  
  
-   Assurez\-vous que la définition d'ensemble d'outils est correcte et que la clé de Registre spécifiée existe.
