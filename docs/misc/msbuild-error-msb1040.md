# Erreur MSBuild MSB1040
**MSB1040: ToolsVersion n'est pas valide.**  
  
 La valeur `ToolsVersion` spécifiée avec le commutateur **\/ToolsVersion** n'est pas valide.  
  
### Pour corriger cette erreur  
  
-   Spécifiez une valeur `ToolsVersion` valide lorsque vous effectuez une génération en utilisant le commutateur **\/ToolsVersion** sur la ligne de commande.
