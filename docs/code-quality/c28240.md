# C28240
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avertissement C28240: L'annotation sur un paramètre possède param2 mais aucun param1.  
  
 Une erreur interne s'est produite dans le fichier de modèle PREfast.  Cet avertissement ne doit pas se produire dans une utilisation classique.
