# C28212
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avertissement C28212 : Expression de pointeur attendu pour l'annotation  
  
 Cet avertissement indique que le paramètre compté à une annotation \(pas la fonction en train d'être annotée\) est attendu comme un pointeur ou un type tableau, mais un autre type a été rencontré.  L'annotation a besoin d'être corrigé.
