# C28211
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avertissement C28211 : Nom du contexte statique attendu pour SAL\_context  
  
 Cet avertissement indique que l'opérande à l'annotation `_Static_context_` doit correspondre au nom d'un contexte défini par un outil.  Cet avertissement ne doit jamais se produire lors d'une utilisation normale.
