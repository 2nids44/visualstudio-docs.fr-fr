# C28229
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avertissement C28229 : L'annotation ne prend pas de paramètres en charge.  
  
 Cet avertissement indique qu'une annotation a été utilisée avec un paramètre lorsque l'annotation est déclarée sans paramètres.  Cela ne doit pas être possible si les macros standard sont utilisées pour les annotations ; cet avertissement n'est pas destiné à être vu dans un usage classique.
