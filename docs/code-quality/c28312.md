# <a name="c28312"></a>C28312
avertissement 28312 : l’annotation < annotation_name > sur la déclaration répétée de \<fonction > \<paramètre > est une version obsolète de SAL.  
  
 L’annotation est une ancienne version et doit être mis à jour à l’équivalent [SAL2](../code-quality/using-sal-annotations-to-reduce-c-cpp-code-defects.md). Cet avertissement n'est pas émis si un avertissement d'annotation non cohérente préalable a été émis. Il est émis une seule fois par déclaration. Inspectez le reste de la déclaration afin de rechercher des SAL obsolètes supplémentaires.
