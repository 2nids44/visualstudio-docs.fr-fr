# <a name="c28280"></a>C28280
avertissement C28280 : pour le symbole, un 'end' a été trouvé sans un correspondant 'begin'

 Prend en charge de la langue de l’annotation de début et de fin (`{` et `}` en C) construction et l’appariement est devenu non équilibré. Cette situation peut être évitée si les macros sont utilisés.
