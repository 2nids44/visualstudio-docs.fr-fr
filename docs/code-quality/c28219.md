# C28219
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avertissement C28219: Membre de l'énumération attendu pour une annotation, le paramètre dans l'annotation  
  
 Il est attendu un paramètre à une annotation soit membre du type nommé `enum` , et un autre symbole a été rencontré ; utilisez un membre de ce type `enum` .  Cela indique généralement une macro d'annotation incorrectement codée.
