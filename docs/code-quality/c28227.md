# <a name="c28227"></a>C28227
Avertissement C28227 : Le Annotation ne peut pas être également un PrimOp (voir la déclaration antérieure)  
  
 Cet avertissement indique que l’annotation nommée est déclarée comme une annotation ordinaire et également a précédemment été déclarée comme un PrimOp. Cela ne doit pas être possible si les macros standards sont utilisées pour les annotations ; Cet avertissement n’est pas censé être consultés en cours d’utilisation classique.
