# <a name="c28301"></a>C28301
avertissement C28301 : aucune annotation pour la première déclaration de \<fonction >.\< Note1 > consultez \<nom de fichier > (\<ligne >). \<Remarque 2 >

 Cet avertissement est émis lorsque l’annotation n’est pas trouvée à la première déclaration d’une fonction donnée.
