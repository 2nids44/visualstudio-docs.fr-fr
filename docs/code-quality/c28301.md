# C28301
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avertissement C28301 : Pas d'annotations pour la première déclaration de \<fonction\>.\<note1\> Voir \<nom\_fichier\>\(\<ligne\>\). \<note2\>  
  
 Cet avertissement est enregistré lorsque les annotations sont introuvables dans la première déclaration d'une fonction donnée.
