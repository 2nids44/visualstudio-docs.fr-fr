# <a name="c28021"></a>C28021
avertissement C28021 : le paramètre \<param > annotées avec \<anno > doit être un pointeur  
  
 Cet avertissement est émis lorsque l’objet annoté n’est pas un type pointeur. Cette annotation ne peut pas être utilisée avec `void` ou les types intégraux.
