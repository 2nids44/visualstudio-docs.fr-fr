# C1253
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avertissement C1253 : Impossible de charger le fichier de modèle.  
  
 L'Outil d'Analyse du code inscrit cet avertissement en cas d'erreur interne dans le fichier modèle, pas dans le code qui est analysé.
