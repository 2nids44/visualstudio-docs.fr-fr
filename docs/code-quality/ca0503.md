# <a name="ca0503"></a>CA0503
La propriété est déconseillée. Utilisez la propriété de remplacement.  
  
 CA0503 d’avertissement est généré lorsqu’une propriété d’analyse de code déconseillée contient une valeur dans le fichier de configuration de build ou de projet. Pour résoudre cet avertissement, utilisez la propriété spécifiée qui la remplace.
