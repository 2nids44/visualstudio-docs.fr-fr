# <a name="c28204"></a>C28204
avertissement C28204 : \<fonction > : seul cette surcharge et celui situé \<nom de fichier > (\<ligne >) sont annotés pour \<paramname > : les deux ou aucune des deux doivent être annotés.

 Cet avertissement est émis lorsqu’il existe une erreur dans les annotations.
