# <a name="c28287"></a>C28287
avertissement C28287 : pour la fonction, erreur de syntaxe dans _supérieure\_() annotation (nom de paramètre non reconnu)  
  
 L’outil d’analyse du Code indique que cet avertissement lorsque le `SAL_at` (`__drv_at`) l’annotation est utilisée et l’expression de paramètre ne peut pas être interprétée dans le contexte actuel. Cela peut inclure à l’aide d’un paramètre mal orthographié ou nom de membre ou une faute d’orthographe de « retourner » ou « this » mots clés.
