# C28223
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Avertissement C28223 : N'a pas trouvé le jeton\/identificateur attendu pour l'annotation, le paramètre  
  
 Cet avertissement indique qu'un paramètre à une annotation devrait être un identificateur, et un autre symbole a été rencontré.  Cela indique généralement une macro d'annotation incorrectement codée.  L'utilisation des mots clés C ou C\+\+ dans cette position provoquera cette erreur.
