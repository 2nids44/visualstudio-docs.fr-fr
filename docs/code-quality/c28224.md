# C28224
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

annotation C28224 : L'annotation requiert des paramètres  
  
 Cet avertissement indique que l'annotation nommée est utilisée sans paramètres et qu'au moins un paramètre est obligatoire.  Cela ne doit pas être possible si les macros standards sont utilisées pour les annotations ; cet avertissement n'est pas destiné à être vu dans le cadre d'une utilisation normale.
