# <a name="c28224"></a>C28224
avertissement C28224 : Annotation requiert des paramètres

 Cet avertissement indique que l’annotation nommée est utilisée sans paramètres, et au moins un paramètre est obligatoire. Cela ne doit pas être possible si les macros standards sont utilisées pour les annotations ; Cet avertissement n’est pas prévu pour être affichés dans une utilisation normale.
