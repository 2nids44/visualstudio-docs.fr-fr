# C28244
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avertissement C28244: L'annotation pour la fonction possède un paramètre\/annotation externe non analysable.  
  
 Cela ne doit actuellement pas être possible si les macros standard sont utilisées pour les annotations ; cet avertissement n'est pas destiné à être vu dans l'utilisation classique.
