# C28209
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avertissement C28209 : La déclaration pour le symbole possède une déclaration en conflit.  
  
 Cet avertissement indique une déclaration d'annotation mal construite.  Cet avertissement ne doit jamais se produire lors d'une utilisation normale.
