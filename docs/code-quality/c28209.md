# <a name="c28209"></a>C28209
avertissement C28209 : la déclaration du symbole possède une déclaration en conflit  
  
 Cet avertissement indique une déclaration d’annotation construit de façon incorrecte. Cet avertissement ne doit jamais se produire en cours d’utilisation normale.
