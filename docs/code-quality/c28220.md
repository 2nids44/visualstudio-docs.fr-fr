# <a name="c28220"></a>C28220
avertissement C28220 : expression d’entier attendue pour l’annotation du paramètre dans l’annotation

 Cet avertissement indique qu’un paramètre à une annotation est censé être une expression d’entier, et un autre type a été rencontrée. Cela indique généralement une macro d’annotation de code incorrect.
