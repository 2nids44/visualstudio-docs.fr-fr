# C28220
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avertissement C28220 : L'expression entière est attendue pour l'annotation au paramètre dans l'annotation  
  
 Cet avertissement indique qu'un paramètre à une annotation est attendu pour être une expression entière, et un autre type a été rencontré.  Cela indique généralement une macro d'annotation incorrectement codée.
