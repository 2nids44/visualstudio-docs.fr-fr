# <a name="ca0055"></a>CA0055
Pas pu charger le fichier :\<chemin d’accès >

 L’outil d’analyse du code n’a pas pu charger le fichier cible qui a été spécifié pour l’analyse. Cela se produit si le fichier est introuvable ou l’accès au fichier a été refusé. En règle générale, cela se produit si l’utilisateur a choisi de substituer la propriété d’assembly d’entrée dans [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)], ou a appelé fxcopcmd.exe manuellement.
