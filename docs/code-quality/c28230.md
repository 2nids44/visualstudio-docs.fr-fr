# <a name="c28230"></a>C28230
avertissement C28230 : le type de paramètre ne possède aucun membre.  
  
 Cet avertissement indique qu’un argument à une annotation tente d’accéder à un `struct`, `class`, ou `union` et le membre nommé n’existe pas.
