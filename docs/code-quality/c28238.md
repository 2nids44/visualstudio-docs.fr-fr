# C28238
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avertissement C28238: l'annotation sur le paramètre possède plus d'une valeur, stringValue, et longValue.  Utiliser paramn \= xxx  
  
 Une erreur interne s'est produite dans le fichier de modèle PREfast.  Cet avertissement ne doit pas se produire dans une utilisation classique.
