# <a name="c28238"></a>C28238
avertissement C28238 : l’annotation sur le paramètre comporte plusieurs value, stringValue et longValue. Utiliser paramn=xxx

 Une erreur interne s’est produite dans le fichier de modèle PREfast. Cet avertissement ne doit pas avoir lieu en cours d’utilisation classique.
