# <a name="ca0053"></a>CA0053
Assembly de règle n’a pas pu être chargé.  
  
 Cela indique qu’un assembly de règle pas pu être chargé par réflexion. Cela peut se produire si l’utilisateur a des règles personnalisées qui n’ont pas été générés correctement, ou l’outil d’analyse ne peut pas charger l’assembly pour d’autres raisons, telles que l’accès au système de fichiers a été refusé.
