# <a name="c28221"></a>C28221
avertissement C28221 : expression de chaîne attendue pour le paramètre dans l’annotation

 Cet avertissement indique qu’un paramètre à une annotation est censé être une chaîne et un autre type a été rencontrée. Généralement, cela indique une macro d’annotation de code incorrect et n’est pas prévu pour être affichés dans une utilisation normale.
