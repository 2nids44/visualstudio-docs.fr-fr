# <a name="c28208"></a>C28208
avertissement C28208 : fonction \<fonction > était précédemment défini avec une autre liste de paramètres à \<fichier > (\<ligne >). Certains outils d’analyse génèrera des résultats incorrects  
  
 Cet avertissement est émis lors de la définition de connu d’une fonction ne correspond pas à une autre occurrence.
