# <a name="c28279"></a>C28279
avertissement C28279 : pour le symbole, un 'begin' a été trouvé sans 'end' correspondant  
  
 Le langage d’annotation prend en charge un début et fin (`{` et `}` en C) construction et l’appariement est devenu non équilibré. Cette situation peut être évitée si les macros sont utilisés.
