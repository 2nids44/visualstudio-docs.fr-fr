# C28286
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avertissement C28286 : Pour la fonction, erreur de syntaxe près de la fin  
  
 L'outil d'analyse du code inscrit cet avertissement lorsqu'une erreur probable s'est produit dans le fichier de modèle.  Certaines erreurs de fichier source peuvent également provoquer de telles erreurs.
