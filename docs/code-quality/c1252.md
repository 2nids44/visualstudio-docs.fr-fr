# <a name="c1252"></a>C1252
avertissement C1252 : une dépendance circulaire ou manquante entre les plug-ins : requiert le GUID  
  
 L’outil d’analyse du Code signale cet avertissement lorsqu’il existe une erreur interne avec des dépendances de plug-in, pas dans le code en cours d’analyse.
