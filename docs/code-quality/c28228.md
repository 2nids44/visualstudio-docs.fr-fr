# C28228
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avertissement C28228 : Paramètre d'annotation : le type ne peut pas être utilisé dans les annotations.  
  
 Cet avertissement indique qu'un paramètre est d'un type qui n'est pas pris en charge.  Les annotations peuvent uniquement utiliser un jeu limité de types en tant que paramètres.  Cela ne doit pas être possible si les macros standard sont utilisées pour les annotations ; cet avertissement n'est pas destiné à être vu dans un usage classique.
