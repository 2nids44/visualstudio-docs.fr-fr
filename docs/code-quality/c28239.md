# C28239
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avertissement C28239 : L'annotation du paramètre a à la fois value, stringValue, ou longValue ; et paramn\=xxx.  Utiliser uniquement paramn\=xxx  
  
 Une erreur interne s'est produite dans le fichier de modèle PREfast.  Cet avertissement ne doit pas se produire dans une utilisation classique.
