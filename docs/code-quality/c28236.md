# <a name="c28236"></a>C28236
avertissement C28236 : l’annotation ne peut pas être utilisée dans une expression  
  
 Cet avertissement indique qu’une tentative a été effectuée pour utiliser une fonction déclarée comme une annotation dans le contexte d’une expression. Cela ne doit pas être possible si les macros standards sont utilisées pour les annotations ; Cet avertissement n’est pas censé être consultés en cours d’utilisation classique.
