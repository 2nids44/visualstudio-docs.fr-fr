# <a name="c28225"></a>C28225
avertissement C28225 : Impossible de trouver le nombre correct de paramètres requis dans l’annotation  
  
 Cet avertissement indique que l’annotation nommée est utilisée avec un nombre incorrect de paramètres. Cela ne doit pas être possible si les macros standards sont utilisées pour les annotations ; Cet avertissement n’est pas censé être consultés en cours d’utilisation classique.
