# C28225
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avertissement C28225: N'a pas trouvé le nombre correct de paramètres requis dans l'annotation  
  
 Cet avertissement indique que l'annotation nommée est utilisée avec un nombre erroné de paramètres.  Cela ne doit pas être possible si les macros standard sont utilisées pour les annotations ; cet avertissement n'est pas destiné à être vu dans l'utilisation classique.
