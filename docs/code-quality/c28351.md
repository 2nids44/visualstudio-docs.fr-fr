# <a name="c28351"></a>C28351
avertissement C28351 : l’annotation \<annotation > décrit où une valeur dynamique (une variable) ne peut pas être utilisée dans la condition.

 Cet avertissement est émis lorsqu’une annotation est appliquée, où le type C/C++ est en cours d’inspection.

 L’expression dans le `_When_` doit évaluer à une constante. Le `_When_` est ignoré.
