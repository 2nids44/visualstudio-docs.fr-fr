# <a name="c28160"></a>C28160
avertissement C28160 : annotation d’erreur

 Cet avertissement est émis lorsqu’un `__drv_error` annotation a été rencontrée. Cette annotation est utilisée pour signaler des pratiques de codage qui doivent être corrigées et peuvent être utilisés avec un `__drv_when` annotation pour indiquer des combinaisons spécifiques de paramètres.
