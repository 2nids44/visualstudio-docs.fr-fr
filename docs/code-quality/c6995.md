# C6995
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avertissement C6995 : Échec de l'enregistrement du fichier journal XML.  
  
 Cet avertissement indique que l'outil d'Analyse du Code ne peut pas créer le journal de défaut, qui est la sortie de l'analyse du code.  
  
 Cette erreur peut indiquer une erreur disque ou indiquer que vous n'avez pas l'autorisation de créer un fichier dans le répertoire spécifié.
