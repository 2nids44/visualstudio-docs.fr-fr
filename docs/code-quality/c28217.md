# C28217
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avertissement C28217 : Pour la fonction, le nombre de paramètres de l'annotation ne correspond pas au nombre trouvé sur le fichier.  
  
 Les annotations sur la ligne en cours et sur la ligne du message sont incohérentes.  Cela ne doit pas être possible si les macros standard sont utilisées pour les annotations ; cet avertissement n'est pas destiné à être vu dans l'utilisation normale.
