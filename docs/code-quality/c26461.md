# <a name="c26461-useconstpointerarguments"></a>C26461 USE_CONST_POINTER_ARGUMENTS :
  L’argument de pointeur 'argument %' pour la fonction 'fonction %' peut être marquée comme un pointeur vers `const`. Consultez [C++ de base des recommandations con.3](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Rconst-ref).
