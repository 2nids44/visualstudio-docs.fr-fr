# C6705
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avertissement C6705 : Les arguments \<expected\_number\> attendus par erreur d'annotation pour l'annotation \<parameter\> ont trouvé \<actual\_number\>.  
  
 Cet avertissement est signalé en cas de erreur dans les annotations.
