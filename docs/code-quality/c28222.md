# <a name="c28222"></a>C28222
avertissement 28222 : _oui\_, _Non\_, ou _Maybe\_ attendue pour l’annotation  
  
 Cet avertissement indique qu’un paramètre à une annotation doit être un des symboles `_Yes_`, `_No_`, ou `_Maybe_`, et certains autres symboles a été rencontrée. Cela indique généralement une macro d’annotation de code incorrect.
