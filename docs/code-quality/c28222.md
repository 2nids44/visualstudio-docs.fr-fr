# C28222
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avertissement 28222 : \_Yes\_, \_No\_, ou \_Maybe\_ prévu pour annotation  
  
 Cet avertissement indique qu'un paramètre d'une annotation devrait être l'un des symboles `_Yes_`, `_No_`, ou `_Maybe_`, et qu'un autre symbole a été rencontré.  Cela indique généralement une macro d'annotation incorrectement codée.
