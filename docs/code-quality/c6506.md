# <a name="c6506"></a>C6506
avertissement C6506 : annotation non valide : \<nom > propriété peut uniquement être employée dans des valeurs de types pointeur ou tableau  
  
 Cet avertissement indique qu’une propriété est utilisée sur un type autre que les types pointeur ou tableau. L’accès, Tainted et propriétés valides peuvent être utilisées sur tous les types de données. Autres propriétés, telles que ValidBytesConst, ValidElementsConst, ElementSize et NullTerminted prennent prend en charge le pointeur, pointeur vers les membres ou types de tableau. Pour obtenir une liste complète des propriétés et les types de données pris en charge, consultez [propriétés de l’Annotation](http://msdn.microsoft.com/en-us/f77b4370-6bda-4294-bd2a-e7d0df182a3d).  
  
## <a name="example"></a>Exemple  
 Le code suivant génère cet avertissement :  
  
```cpp  
#include<sal.h>  
void f(_Out_ char c)  
{  
    c = 'd';  
}  
```  
  
 Pour corriger cet avertissement, utilisez un pointeur ou un type de tableau, comme indiqué dans l’exemple de code suivant :  
  
```cpp  
#include<sal.h>  
void f(_Out_ char *c)  
{  
    *c = 'd';  
}  
```
