# C28275
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avertissement C28275 : le paramètre de \_Macro\_value\_ a la valeur null  
  
 Cet avertissement indique qu'il existe une erreur interne dans le fichier de modèle, pas dans le code qui est analysé.  La fonction *macroValue* a été appelée sans paramètre.
