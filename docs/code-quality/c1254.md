# C1254
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avertissement C1254 : Incompatibilité de versions de plug\-ins : la version ne correspond pas à la version du pilote PREfast.  
  
 L'outil d'analyse du code signale cet avertissement en cas d'erreur interne avec la version du plug\-in, pas dans le code qui est analysé.
