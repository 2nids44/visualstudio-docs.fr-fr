# <a name="c1254"></a>C1254
avertissement C1254 : incompatibilité de version de plug-in : version ne correspond pas à la version du pilote PREfast

 L’outil d’analyse du Code signale cet avertissement lorsqu’il existe une erreur interne avec la version du plug-in, pas dans le code en cours d’analyse.
