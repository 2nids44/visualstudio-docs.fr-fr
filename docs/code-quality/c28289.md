# C28289
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avertissement C28289 : Pour la fonction : ReadableTo ou WritableTo n'a pas eu de spécification de limites en tant que paramètre.  
  
 L'outil d'Analyse du Code inscrit cet avertissement lorsque l'annotation de la fonction \/ du paramètre est mal codé comme indiqué.
