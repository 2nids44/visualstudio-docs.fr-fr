# C28232
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avertissement C28232 : Pre, Post, ou Deref appliqué à aucune annotation.  
  
 Cet avertissement indique qu'un opérateur`_Pre_`, `_Post_`, ou `_Deref_` apparaît dans une expression d'annotation sans annotation fonctionnelle ultérieure ; le modificateur a été ignoré, mais cela indique une annotation codée de manière incorrecte.
