# C28272
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avertissement C28272 : L'annotation pour la fonction, paramètre lors de l'examination est incohérente avec la déclaration de la fonction.  
  
 Cet avertissement indique une erreur dans les annotations, pas dans le code qui est en train d'être analysé.  Les annotations apparaissant dans une définition de fonction sont incohérentes avec celles qui apparaissent dans une déclaration.  Les deux annotations devraient être résolues pour correspondre.
