# <a name="c1256"></a>C1256
avertissement C1256 : PCH doit être écrites et lues.  
  
 L’outil d’analyse du Code signale cet avertissement lorsqu’il existe une erreur interne dans l’outil, et non dans le code en cours d’analyse.
