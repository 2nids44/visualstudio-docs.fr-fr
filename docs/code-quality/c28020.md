# C28020
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avertissement C28020 : L'expression \<expr\> n'a pas la valeur true dans cet appel.  
  
 Cet avertissement est enregistré lorsque l'expression \_Satisfies\_ répertoriée n'est pas vraie.  Fréquemment cela indique un paramètre incorrect.  
  
 Si cela se produit dans une déclaration de fonction, les annotations indiquent une condition impossible.
