# <a name="ca0057"></a>CA0057
Les violations ne peut pas être mappées à l’ensemble actuel de cibles et de règles

 Cette erreur se produit si FxCopCmd.exe est appelé manuellement avec un fichier de projet ou un rapport XML qui comporte des violations qui ne peut pas être mappées à l’ensemble actuel de cibles et de règles importé.

 Se produit uniquement si l’analyse est effectuée à l’aide de la version de ligne de commande de l’outil.
