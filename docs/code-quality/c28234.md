# C28234
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avertissement C28234 : L'expression \_at ne s'applique pas à la fonction courante.  
  
 Cet avertissement indique que la valeur d'une expression `_At_` ne reconnaît pas un objet accessible.
