# C28285
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avertissement C28285 : Pour la fonction, erreur de syntaxe dans le paramètre  
  
 L'outil d'analyse du code inscrit cet avertissement lorsqu'une erreur probable s'est produite dans le fichier modèle.  Certaines erreurs de fichier source peuvent également provoquer de telles erreurs.
