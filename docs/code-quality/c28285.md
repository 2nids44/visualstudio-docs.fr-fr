# <a name="c28285"></a>C28285
avertissement C28285 : pour la fonction, erreur de syntaxe dans le paramètre  
  
 L’outil d’analyse du Code indique que cet avertissement lorsqu’une erreur est rencontrée dans le fichier de modèle. Certaines erreurs de fichier source peuvent également provoquer des erreurs.
