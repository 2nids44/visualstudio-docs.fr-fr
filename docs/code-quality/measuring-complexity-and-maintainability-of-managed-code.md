# <a name="measuring-complexity-and-maintainability-of-managed-code"></a>Mesures de la complexité et de la facilité de maintenance du code managé
La complexité accrue d’applications modernes augmente également la difficulté de rendre le code fiable et facile à gérer. Ces dernières années, de mesures de logiciel, appelées la métrique du code, ont été développés peut aider les développeurs à comprendre où leur code doit reprendre ou test accrue.  
  
 Les développeurs peuvent utiliser [!INCLUDE[vsprvsts](../code-quality/includes/vsprvsts_md.md)] pour générer des données de métrique du code qui mesurent la complexité et la facilité de maintenance de leur code managé. Données de métrique du code peuvent être générées pour une solution entière ou un seul projet.  
  
 Cette section décrit comment générer et utiliser la métrique du code dans [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)].
