# Proc&#233;dure&#160;: ajouter des nœuds &#224; l&#39;espace de travail &#224; partir de la vue de d&#233;part
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Cette rubrique montre comment ajouter des nœuds globaux à l'[Espace de travail du Concepteur de schémas XML](../xml-tools/xml-schema-designer-workspace.md) à partir de la [vue de départ](../xml-tools/start-view.md).  
  
### Pour ajouter des nœuds à l'espace de travail à partir de la vue de départ  
  
1.  Suivez la procédure détaillée dans [Procédure : créer et modifier un fichier de schéma XSD](../xml-tools/how-to-create-and-edit-an-xsd-schema-file.md).  
  
2.  Cliquez avec le bouton droit dans l'Éditeur XML et sélectionnez **Concepteur de vues**.  
  
     La vue de départ apparaît.  
  
     Dans la section **Détails du jeu de schémas**, notez la présence de deux \(**2**\) **Types complexes globaux**.  
  
3.  Cliquez sur le lien **ajouter** à droite du titre **Types complexes globaux**.  
  
     Les nœuds de type complexe `USAddress` et `PurchaseOrderType` apparaissent sur l'aire de conception de la vue du graphique.
