# How to: Print Class Diagrams (Class Designer)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Vous pouvez imprimer un diagramme de classes à l'aide de la fonctionnalité d'impression de Visual Studio.  
  
### Pour imprimer un diagramme de classes  
  
1.  Affichez le diagramme de classes.  \(Si nécessaire, sélectionnez l'onglet Diagramme de classes pour l'afficher.\)  
  
2.  Dans le menu **Fichier**, cliquez sur **Imprimer**.  
  
     Le diagramme de classes sera imprimé dans sa totalité.  Notez que vous devrez peut\-être ajuster les paramètres de la boîte de dialogue Mise en page pour que l'impression ait la taille appropriée.
