# How to: Add Comments to Class Diagrams (Class Designer)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Vous pouvez utiliser les zones de commentaire pour annoter les diagrammes de classes.  Une zone de commentaire possède une propriété, **Texte**, dans lequel vous pouvez taper le texte.  Les zones de commentaire existent uniquement sur la surface du diagramme, et pas dans le code.  
  
 Un commentaire réside sur la vue du diagramme de classes  du **Concepteur de classes** ; si vous ouvrez un deuxième diagramme sur le même projet, les commentaires créés dans la première vue ne sont pas visibles.  Si vous supprimez un diagramme, tous les commentaires qu'il contenait sont également supprimés.  
  
 Vous pouvez redimensionner une zone de commentaire, mais ne pouvez pas modifier d'autres aspects de son apparence, comme sa couleur d'arrière\-plan, sa police ou sa taille de police.  
  
### Pour ajouter un commentaire  
  
1.  Faites glisser un commentaire de la **boîte à outils du Concepteur de classes** vers le diagramme de classes.  
  
2.  Cliquez dans la nouvelle zone de commentaire du diagramme et tapez le texte de votre choix.
