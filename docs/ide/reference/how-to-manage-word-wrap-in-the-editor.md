# Comment&#160;: g&#233;rer le retour automatique &#224; la ligne dans l&#39;&#201;diteur
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

Vous pouvez définir l'option **Retour automatique à la ligne**, ainsi que la désactiver.  Lorsque cette option est définie, le fragment d'une ligne s'étendant au\-delà de la largeur courante de la fenêtre de l'éditeur de code s'affiche sur la ligne suivante.  Quand cette option est désactivée, par exemple pour simplifier l'utilisation de la numérotation des lignes, vous pouvez faire défiler l'écran vers la droite pour afficher la fin d'une ligne excédant la largeur de l'écran.  
  
 Pour plus d'informations, consultez [How to: Set General Editor Options](http://msdn.microsoft.com/fr-fr/704e4a7b-2162-4bed-8a47-f4f6ffec98c2).  
  
> [!NOTE]
>  Les boîtes de dialogue et les commandes de menu que vous voyez peuvent différer de celles qui sont décrites dans l'**aide** selon vos paramètres actifs ou votre édition.  Pour modifier vos paramètres, choisissez **Importation et exportation de paramètres** dans le menu **Outils**.  Pour plus d'informations, consultez [Customizing Development Settings in Visual Studio](http://msdn.microsoft.com/fr-fr/22c4debb-4e31-47a8-8f19-16f328d7dcd3).  
  
## Procédure  
  
#### Pour définir les préférences de retour automatique à la ligne  
  
1.  Dans le menu **Outils**, sélectionnez **Options**.  
  
2.  Dans le dossier **Éditeur de texte**, choisissez les options **Général** du sous\-dossier **Tous les langages** pour définir cette option de manière globale.  
  
     – ou –  
  
     Choisissez les options **Général** du sous\-dossier du langage dans lequel vous programmez.  
  
3.  Sous **Paramètres**, activez ou désactivez l'option **Retour automatique à la ligne**.  
  
     Lorsque l'option **Retour automatique à la ligne** est sélectionnée, l'option **Afficher des glyphes visuels pour le retour automatique à la ligne** est activée.  
  
4.  Sélectionnez l'option **Afficher des glyphes visuels pour le retour automatique à la ligne** si vous préférez afficher une flèche retour quand une ligne nécessite un retour automatique à la ligne suivante.  Désactivez cette option si vous préférez ne pas afficher de flèche indicatrice.  
  
    > [!NOTE]
    >  Ces flèches de rappel ne s'ajoutent pas à votre code : elles ne sont là qu'à des fins d'affichage.
