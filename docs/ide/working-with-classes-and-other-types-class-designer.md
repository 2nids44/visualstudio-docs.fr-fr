# <a name="working-with-classes-and-other-types-class-designer"></a>Utilisation des classes et d'autres types (Concepteur de classes)
Concentrez-vous plus facilement sur les classes et les types qui vous intéressent le plus, quand vous les créez et les changez à l'aide du Concepteur de classes dans Visual Studio. Utilisez des diagrammes de classes pour travailler de manière plus visuelle sur votre code Visual C# .NET, Visual Basic .NET ou C++.
