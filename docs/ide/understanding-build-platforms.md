# Pr&#233;sentation des plateformes de g&#233;n&#233;rations
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Vous pouvez stocker les différentes versions de la solution et les propriétés de projet qui s'appliquent à différentes plateformes cibles.  Par exemple, vous pouvez créer une configuration de débogage qui cible une plateforme x86 et une configuration debug qui vise une plateforme x64.  Vous pouvez rapidement modifier la plateforme active afin que vous puissiez facilement générer plusieurs configurations.
