# Comment&#160;: modifier des requ&#234;tes TableAdapter
Vous modifiez des requêtes TableAdapter à l'aide du [Configuration de requête TableAdapter \(Assistant\)](../data-tools/editing-tableadapters.md) dans le **Concepteur de DataSet**.  Vous devez modifier une requête TableAdapter si elle n'est plus adaptée aux besoins de votre application.  \(Vous pouvez également créer des requêtes supplémentaires sur le TableAdapter.  Pour plus d'informations sur l'ajout de nouvelles requêtes, consultez [Comment : créer des requêtes TableAdapter](../data-tools/how-to-create-tableadapter-queries.md).\)  
  
> [!NOTE]
>  Si le [Configuration de TableAdapter \(Assistant\)](../Topic/TableAdapter%20Configuration%20Wizard.md) s'ouvre au lieu de l'**Assistant Configuration de requêtes TableAdapter**, vous avez peut\-être sélectionné la requête `Fill` principale du TableAdapter au lieu d'une des requêtes supplémentaires du TableAdapter.  Pour plus d'informations sur la modification de la requête `Fill` principale du TableAdapter, consultez [Comment : modifier des TableAdapters](../Topic/How%20to:%20Edit%20TableAdapters.md).  
  
 ![TableAdapter avec plusieurs requêtes](../data-tools/media/tableadapter.gif "TableAdapter")  
  
### Pour modifier une requête TableAdapter  
  
1.  Ouvrez le groupe de données dans le **Concepteur de DataSet**.  Pour plus d'informations, consultez [Comment : ouvrir un groupe de données dans le Concepteur de DataSet](../Topic/How%20to:%20Open%20a%20Dataset%20in%20the%20Dataset%20Designer.md).  
  
2.  Sélectionnez la requête TableAdapter à modifier.  
  
3.  Cliquez avec le bouton droit sur la requête TableAdapter et sélectionnez **Configurer**.  
  
     L'**Assistant Configuration de requêtes TableAdapter** s'ouvre afin de vous permettre de modifier la requête ou la procédure stockée pour cette requête.  
  
4.  Exécutez l'**Assistant Configuration de requêtes TableAdapter** en apportant les modifications de votre choix.  Pour plus d'informations, consultez [Configuration de requête TableAdapter \(Assistant\)](../data-tools/editing-tableadapters.md).
