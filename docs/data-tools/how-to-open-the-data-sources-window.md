# Comment&#160;: ouvrir la fen&#234;tre Sources de donn&#233;es
La fenêtre **Sources de données** affiche les sources de données contenues dans votre projet.  Utilisez la fenêtre **Sources de données** pour créer une interface utilisateur constituée de contrôles liés aux données en faisant glisser des éléments depuis cette fenêtre jusqu'à des formulaires de votre projet.  Consultez [Liaison de contrôles Windows Forms à des données dans Visual Studio](../data-tools/bind-windows-forms-controls-to-data-in-visual-studio.md).  
  
 Vous pouvez remplir la fenêtre **Sources de données** en exécutant l'Assistant **Configuration de source de données**.  Pour plus d'informations, consultez [Configuration de source de données \(Assistant\)](../data-tools/media/data-source-configuration-wizard.png).  
  
### Pour ouvrir la fenêtre Sources de données  
  
-   Dans la barre de menus, choisissez **Affichage**, **Autres fenêtres**, **Sources de données** \(ou appuyez sur les touches Maj\+Alt\+D\).  
  
    > [!IMPORTANT]
    >  Vous pouvez ajouter une source de données à un projet uniquement s'il prend en charge la création et l'utilisation des sources de données.  Par exemple, vous ne pouvez pas ouvrir la fenêtre **Sources de données** dans un projet pour une application Windows Store.
