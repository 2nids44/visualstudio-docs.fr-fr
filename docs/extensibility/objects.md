# d&#39;entreprise
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Les informations de référence suivantes expliquent les objets qui sont utilisés pour créer une instance de l'éditeur de texte principal.  
  
## Dans cette section  
 [Objet de VSCodeWindow](../extensibility/vscodewindow-object.md)  
 Décrit l'objet de fenêtre de code, une fenêtre de document spécialisées pouvant contenir une ou plusieurs vues de texte.  
  
 [VSCodeWindowManager objet](../extensibility/vscodewindowmanager-object.md)  
 Décrit l'objet de gestionnaire de fenêtre de code qui est implémentée par le service de langage et est chargé de gérer les motifs.  
  
 [Objet de VSTextBuffer](../extensibility/vstextbuffer-object.md)  
 Décrit l'objet de mémoire tampon de texte, qui représente un flux de texte Unicode, les données d'un fichier.  
  
 [Objet de VSTextView](../extensibility/vstextview-object.md)  
 Décrit l'affichage de texte, une fenêtre qui permet aux utilisateurs permet d'afficher et de modifier du texte Unicode de la mémoire tampon de texte.
