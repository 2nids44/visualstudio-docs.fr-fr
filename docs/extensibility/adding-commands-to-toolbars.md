# Ajouter des commandes aux barres d&#39;outils
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Une barre d'outils est une barre horizontale ou verticale qui comporte des boutons liés aux commandes. Vous pouvez ajouter vos propres commandes aux barres d'outils dans de nombreuses fenêtres différentes dans Visual Studio : la fenêtre principale de Visual Studio, l'Explorateur de solutions et vos propres fenêtres Outil personnalisées.  
  
 Cette section présente un certain nombre de choses que vous pouvez faire avec les commandes des barres d'outils.
