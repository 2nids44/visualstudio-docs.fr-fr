# <a name="extensibility-in-visual-studio"></a>Extensibilité de Visual Studio
Visual Studio vous fournit de nombreuses options d’extensibilité. Vous pouvez créer vos propres kits de développement logiciel, utilisez Windows, Windows Phone, les kits de développement logiciel Azure (qui sont installés dans le cadre de Visual Studio) et étendre Visual Studio.  
  
## <a name="extend-visual-studio"></a>Extension de Visual Studio  
 Vous pouvez utiliser le Kit de développement Visual Studio pour étendre chaque partie de Visual Studio : commandes, menus, windows, les éditeurs et projets. Pour en savoir plus, consultez [Visual Studio SDK](../extensibility/visual-studio-sdk.md).  
  
## <a name="create-your-own-sdks"></a>Créer vos propres kits de développement logiciel  
 Découvrez comment créer, empaqueter et déployer vos propres kits de développement logiciel platform et d’extension : [création d’un Kit de développement logiciel](../extensibility/creating-a-software-development-kit.md).  
  
## <a name="roslyn-extensibility"></a>Extensibilité de Roslyn  
 Découvrez comment utiliser Roslyn pour l’extensibilité : [.NET Compiler Platform (« Roslyn ») extensibilité](../extensibility/dotnet-compiler-platform-roslyn-extensibility.md).
