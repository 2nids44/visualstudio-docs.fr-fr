# <a name="color-theming-tools"></a>Outils de thèmes de couleurs
Le Kit de développement logiciel Visual Studio fournit plusieurs outils pour vous aider à créer et gérer des thèmes de couleurs.  
  
-   Le [de couleurs VSIX](../../extensibility/internals/vsix-color-editor.md) outil peut créer et modifier des couleurs personnalisées pour Visual Studio.  
  
-   Le [VSIX couleur compilateur](../../extensibility/internals/vsix-color-compiler.md) outil convertit les fichiers .xml de thème Visual Studio existants en .pkgdef pour votre extension.
