# <a name="outlining-and-hidden-text-in-a-legacy-language-service"></a>Texte en mode plan et masqué dans un Service de langage hérité
Le mode plan permet de réduire une séquence de lignes de texte en une seule ligne. Par exemple, dans C++, toutes les méthodes peuvent être réduites à une seule ligne, n'indiquant que la signature de méthode. Texte masqué est la séquence de lignes de texte qui peuvent être affichés ou masqués.
