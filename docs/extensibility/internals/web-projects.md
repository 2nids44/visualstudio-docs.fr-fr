# <a name="web-projects"></a>Projets web
Un système de projet Web est un système de projet qui crée des projets Web. À leur tour, les projets Web créent des applications Web. Vous pouvez programmer une page Web à l’aide de langages de programmation traditionnels, tels que [!INCLUDE[vbprvb](../../code-quality/includes/vbprvb_md.md)] ou [!INCLUDE[csprcs](../../data-tools/includes/csprcs_md.md)], pour collecter et traiter les informations d’un utilisateur, stockez-le dans une base de données et ainsi de suite. Pages Web avec le code associé sont appelés des pages Web actives.
