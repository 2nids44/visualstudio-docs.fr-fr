# <a name="runtime-activity-designers"></a>Concepteurs d'activités d'exécution

Les rubriques contenues de cette section fournissent des conseils sur la façon d'utiliser les concepteurs d'activités [!INCLUDE[wfd2](../workflow-designer/includes/wfd2_md.md)].
