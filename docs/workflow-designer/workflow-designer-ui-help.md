# <a name="workflow-designer-ui-help"></a>Aide sur l'interface utilisateur de Workflow Designer
Cette section contient les rubriques d’aide contextuelle pour Windows Workflow Designer. Lorsque vous travaillez dans [!INCLUDE[vs2010](../misc/includes/vs2010_md.md)], vous pouvez accéder à ces rubriques à partir de l’interface utilisateur en appuyant sur **F1** lorsqu’un [!INCLUDE[wfd2](../workflow-designer/includes/wfd2_md.md)] boîte de dialogue est ouverte.

 Cette section contient également plusieurs rubriques concernant la navigation et les messages d'erreur.
