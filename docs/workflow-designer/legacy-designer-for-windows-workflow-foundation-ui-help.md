# <a name="legacy-designer-for-windows-workflow-foundation-ui-help"></a>Aide de l'interface utilisateur du concepteur hérité pour Windows Workflow Foundation

Cette section contient les rubriques d’aide contextuelle pour le Concepteur de flux de travail Windows hérité dans Visual Studio 2010. Utilisez le concepteur hérité lorsque vous avez besoin cibler le .NET Framework version 3.5 ou le WinFX.

Vous pouvez accéder à ces rubriques à partir de l’interface utilisateur de Visual Studio en appuyant sur **F1** lorsqu’une boîte de dialogue Concepteur de Workflow héritée est ouverte.
