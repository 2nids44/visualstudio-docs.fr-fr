# Th&#232;mes, Concepteur de Workflow, Options, bo&#238;te de dialogue (h&#233;rit&#233;e)
Cette rubrique explique comment utiliser la boîte de dialogue **Thèmes, Concepteur de Workflow, Options** dans le [!INCLUDE[wfd1](../workflow-designer/includes/wfd1_md.md)] hérité.Utilisez le [!INCLUDE[wfd2](../workflow-designer/includes/wfd2_md.md)] hérité lorsque vous devez cibler le [!INCLUDE[netfx35_long](../workflow-designer/includes/netfx35_long_md.md)] ou le [!INCLUDE[vstecwinfx](../workflow-designer/includes/vstecwinfx_md.md)].  
  
 Un thème définit les couleurs d'arrière\-plan et de premier plan, les styles, les icônes et les autres éléments visuels d'un workflow.Vous pouvez enregistrer des thèmes en vue de les réutiliser pour d'autres workflows.Vous pouvez sélectionner les thèmes par défaut et des thèmes personnalisés enregistrés via la boîte de dialogue **Thèmes, Concepteur de flux de travail, Options**.Pour ouvrir cette boîte de dialogue, cliquez sur **Sélectionner un thème personnalisé** dans le menu **Workflow**.Vous pouvez également cliquer avec le bouton droit sur l'aire de conception de workflow et cliquez sur **Sélectionner un thème personnalisé**.  
  
 Le tableau suivant décrit les éléments d'interface utilisateur de la boîte de dialogue **Thèmes, Concepteur de flux de travail, Options**.  
  
|Élément d'interface|Description|  
|-------------------------|-----------------|  
|**Nouveau**|Permet d'ouvrir la [Configuration du thème, boîte de dialogue \(héritée\)](../workflow-designer/theme-configuration-dialog-box-legacy.md) pour créer un nouveau thème.|  
|**Modifier...**|Permet d'ouvrir la [Configuration du thème, boîte de dialogue \(héritée\)](../workflow-designer/theme-configuration-dialog-box-legacy.md) pour modifier le thème sélectionné.|  
|**Parcourir**|Permet de rechercher un fichier de thème de workflow, qui utilise une extension de nom de fichier .wtm.|  
|**Sélectionner un thème**|En\-tête de colonne de nom de thème au sein de listes de thèmes.Sélectionnez un thème dans la liste, puis cliquez sur **Modifier** pour le modifier ou sur **OK** si vous souhaitez utiliser le thème sélectionné pour le workflow.|  
|**Description**|En\-tête de colonne de description de thèmes au sein de listes de thèmes.|  
|**Emplacement**|En\-tête de colonnes d'emplacement de thèmes au sein de listes de thèmes.|
