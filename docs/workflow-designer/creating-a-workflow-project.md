# <a name="creating-a-workflow-project"></a>Création d'un projet de workflow

Flux de travail, les services Windows Communication Foundation (WCF), des activités personnalisées et concepteurs d’activités personnalisées peuvent être créés à l’aide de modèles de projet Visual Studio. Cette section décrit comment créer des bibliothèques et des applications avec les modèles de projet disponibles dans Visual Studio.

Les quatre premières rubriques ci-dessous décrivent comment utiliser des modèles de Workflow sur le **nouveau projet** fenêtre. Ces modèles sont accessibles en sélectionnant dans Visual Studio 2010 **fichier** > **nouveau** > **projet**, puis le  **Flux de travail** catégorie à partir de la **modèles installés** section sur la gauche, sous la **Visual Basic** ou **Visual C#** nœud.

La dernière rubrique décrit comment ajouter des activités de flux de travail, les concepteurs et les services ou autre projet d’éléments à un flux de travail existant (ou autre) de Visual Studio.
