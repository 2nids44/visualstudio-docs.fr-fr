# How to: Specify a Link for Technical Support
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Lors de la publication d'une application [!INCLUDE[ndptecclick](../deployment/includes/ndptecclick_md.md)], la propriété **URL du support technique** identifie une page Web ou un partage de fichiers où les utilisateurs peuvent aller chercher des informations relatives à l'application.  Cette propriété est facultative ; si elle est définie, l'URL sera affichée dans la boîte de dialogue **Ajouter ou supprimer des programmes** de l'entrée de l'application.  
  
 La propriété **URL du support technique** peut être définie dans la page **Publier** du **Concepteur de projets**.  
  
### Pour spécifier une URL de support technique  
  
1.  Un projet étant sélectionné dans l'**Explorateur de solutions**, cliquez dans le menu **Projet** sur **Propriétés**.  
  
2.  Cliquez sur l'onglet **Publier**.  
  
3.  Cliquez sur le bouton **Options** pour ouvrir la boîte de dialogue **Options de publication**.  
  
4.  Cliquez sur **Description**.  
  
5.  Dans le champ **URL du support technique**, entrez le chemin qualifié complet d'un site Web, d'une page Web ou d'un partage UNC.
