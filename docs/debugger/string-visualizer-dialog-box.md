# Visualiseur de cha&#238;ne, bo&#238;te de dialogue
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Cette boîte de dialogue est l'un des visualiseurs standard inclus avec le débogueur Visual Studio.  Outre le texte normal, le visualiseur de chaîne peut évaluer les chaînes HTML et XML, ce qui vous permet d'afficher les résultats dans un format significatif.
