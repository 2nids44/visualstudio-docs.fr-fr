# <a name="debug-universal-windows-apps-uwp"></a>Déboguer des applications de plateforme Windows universelle (UWP)
Le débogueur Visual Studio fournit la prise en charge complète pour le débogage d’applications UWP.  
  
Pour plus d’informations sur le débogage des applications Windows universelles, consultez [déploiement et débogage des applications UWP](/windows/uwp/debug-test-perf/deploying-and-debugging-uwp-apps).  
  
Pour déboguer des applications Windows Phone 8, consultez [débogage des applications pour Windows Phone 8](https://msdn.microsoft.com/en-us/library/windows/apps/ff402572(v=vs.105).aspx).
