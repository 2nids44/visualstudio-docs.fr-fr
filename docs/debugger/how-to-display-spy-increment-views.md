# How to: Display Spy++ Views
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Lorsque Spy\+\+ démarre, il s'ouvre dans la vue [Fenêtres](../debugger/windows-view.md), une arborescence de toutes les fenêtres et de tous les contrôles disponibles sur votre ordinateur.  Cette fenêtre racine a pour titre "Fenêtres 1". Il existe également trois autres vues Spy\+\+ disponibles : la [vue Messages](../debugger/messages-view.md), la [vue Processus](../debugger/processes-view.md) et la [vue Threads](../debugger/threads-view.md).  Vous pouvez afficher la vue souhaitée dans le menu **Espionner**.  Pour plus d'informations, consultez [Vues Spy\+\+](../debugger/spy-increment-views.md).
