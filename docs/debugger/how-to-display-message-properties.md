# How to: Display Message Properties
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Pour plus d'informations sur un message listé dans la [vue Messages](../debugger/messages-view.md), utilisez la [boîte de dialogue Propriétés du message](../debugger/message-properties-dialog-box.md).  
  
### Pour ouvrir une boîte de dialogue Propriétés du message  
  
1.  Sélectionnez un message dans l'arborescence affichée dans la vue Messages.  
  
2.  Dans le menu **Affichage**, sélectionnez **Propriétés**.  
  
 Les boîtes de dialogue Propriétés ne sont pas modales. Ainsi, si vous cliquez sur un autre élément dans une fenêtre d'affichage, la boîte de dialogue affichera des informations sur l'élément sélectionné.
