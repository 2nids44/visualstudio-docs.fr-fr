# How to: Start and Stop the Message Log Display
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Lorsqu'une fenêtre [Vue Messages](../debugger/messages-view.md) est active, l'option **Démarrer l'enregistrement dans le journal** ou **Arrêter l'enregistrement** apparaît dans le menu **Messages** de Spy\+\+ ; en outre, le bouton de barre d'outils **Démarrer l'enregistrement dans le journal** ou **Arrêter l'enregistrement** devient actif.  
  
### Pour démarrer l'affichage du journal des messages  
  
1.  Dans le menu **Messages**, choisissez **Démarrer l'enregistrement dans le journal**.  
  
### Pour arrêter l'affichage du journal des messages  
  
1.  Dans le menu **Messages**, choisissez **Arrêter l'enregistrement**.
