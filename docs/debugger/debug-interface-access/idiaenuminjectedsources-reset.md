# <a name="idiaenuminjectedsourcesreset"></a>IDiaEnumInjectedSources::Reset
Réinitialise la séquence d’énumération au début.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
HRESULT Reset();  
```  
  
## <a name="return-value"></a>Valeur de retour  
 Retourne `S_OK`.
