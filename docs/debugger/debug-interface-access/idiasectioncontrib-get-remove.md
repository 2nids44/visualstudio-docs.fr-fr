# <a name="idiasectioncontribgetremove"></a>IDiaSectionContrib::get_remove
Récupère un indicateur qui indique si la section est supprimée avant qu’il est mis à la partie de l’image en mémoire.  
  
## <a name="syntax"></a>Syntaxe  
  
```C++  
HRESULT get_remove (   
   BOOL* pRetVal  
);  
```  
  
#### <a name="parameters"></a>Paramètres  
 `pRetVal`  
 [out] Retourne `TRUE` si la section ne doit ne pas être ajouté à l’image en mémoire ; sinon, retourne `FALSE`.  
  
## <a name="return-value"></a>Valeur de retour  
 En cas de réussite, retourne `S_OK`. Retourne `S_FALSE` si cette propriété n’est pas pris en charge. Sinon, retourne un code d'erreur.
