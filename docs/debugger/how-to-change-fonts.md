# How to: Change Fonts
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Vous pouvez modifier la police, le style de police et la taille de police des fenêtres Spy\+\+.  
  
### Pour modifier les options de police  
  
1.  Dans le menu **Affichage**, sélectionnez **Police**.  
  
2.  Choisissez une police, un style de police et une taille de police.  
  
3.  Cliquez sur **OK**.  
  
 Si vous activez la case à cocher **Enregistrer la police comme police par défaut**, toutes les fenêtres ultérieures de Spy\+\+ utiliseront cette police.
