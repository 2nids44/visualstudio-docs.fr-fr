# How to: Start Spy++
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Vous pouvez démarrer Spy\+\+ dans Visual Studio ou à partir d'une invite de commandes.  
  
 Lorsque vous démarrez Spy\+\+, si un message s'affiche pour demander l'autorisation d'effectuer des modifications sur l'ordinateur, cliquez sur **Oui**.  
  
> [!NOTE]
>  Vous ne pouvez exécuter qu'une seule instance de Spy\+\+.  Si vous essayez d'exécuter une autre instance, en fait l'instance en cours d'exécution devient active.  
  
### Pour démarrer Spy\+\+ à partir de Visual Studio  
  
-   Dans le menu **Outils**, cliquez sur **Spy\+\+**.  
  
     Comme Spy\+\+ s'exécute en mode autonome, une fois qu'il a démarré, vous pouvez fermer Visual Studio.  
  
    > [!NOTE]
    >  Lorsque vous consignez des messages avec Spy\+\+, l'opération peut ralentir le système d'exploitation.  
  
### Pour démarrer Spy\+\+ à partir d'une invite de commandes  
  
1.  Dans une fenêtre d'invite de commandes, changez de répertoire et passez dans le dossier qui contient spyxx.exe.  En général, le chemin d'accès de ce dossier est ..  \\*Dossier d'installation Visual Studio*\\Common7\\Tools\\.  
  
2.  Tapez **spyxx.exe**, puis appuyez sur Entrée.
