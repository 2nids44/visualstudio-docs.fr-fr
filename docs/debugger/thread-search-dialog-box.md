# Bo&#238;te de dialogue Recherche d&#39;un thread
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Utilisez cette boîte de dialogue pour rechercher et sélectionner le nœud d'un thread spécifique dans la [vue Threads](../debugger/threads-view.md).  Pour afficher cette boîte de dialogue, déplacez le focus sur une fenêtre **Vue Threads**.  Dans le menu **Rechercher**, choisissez **Rechercher le thread**.  
  
 Les paramètres suivants sont disponibles :  
  
 **Thread**  
 ID de thread à rechercher.  
  
 **Module**  
 Chaîne de module à rechercher.  
  
 **Sens de la recherche \(vers le haut ou le bas\)**  
 Direction initiale de la recherche.
