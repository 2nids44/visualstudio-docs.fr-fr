# WPF .Targets Files
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

[!INCLUDE[TLA#tla_winclient](../misc/includes/tlasharptla_winclient_md.md)] étend le [!INCLUDE[TLA#tla_msbuild](../msbuild/includes/tlasharptla_msbuild_md.md)] en ajoutant un jeu de tâches spécifiques à [!INCLUDE[TLA2#tla_wpf](../msbuild/includes/tla2sharptla_wpf_md.md)] combinées dans un fichier .targets spécifique, **Microsoft.WinFX.targets**.  Ce fichier combine le jeu de tâches [!INCLUDE[TLA2#tla_msbuild](../msbuild/includes/tla2sharptla_msbuild_md.md)] qui sont requises pour générer un projet [!INCLUDE[TLA2#tla_msbuild](../msbuild/includes/tla2sharptla_msbuild_md.md)] dans [!INCLUDE[TLA#tla_winclient](../misc/includes/tlasharptla_winclient_md.md)].
