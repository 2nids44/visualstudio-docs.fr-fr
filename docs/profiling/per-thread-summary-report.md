# R&#233;sum&#233; par thread (rapport)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Cet histogramme présente la proportion de temps que chaque thread caché passe dans chaque catégorie d'activité pendant la plage horaire actuellement visible. "Exécution" signifie que le thread est en cours d'exécution ; toutes les autres catégories impliquent que le thread attend une opération.
