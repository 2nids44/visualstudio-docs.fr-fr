# Copier la s&#233;lection
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Pour copier la pile d'appels entière depuis l'onglet Rapport, cliquez sur **Copier**.  Vous pouvez ensuite coller la pile d'appels dans n'importe quel programme prenant en charge cette opération.
