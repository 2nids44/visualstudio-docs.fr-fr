# Interaction de couche, vues
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Le profilage des interactions de couche fournit des informations supplémentaires sur les temps d'exécution des fonctions dans les applications multicouches qui communiquent avec les bases de données via [!INCLUDE[vstecado](../data-tools/includes/vstecado_md.md)].  Les données sont collectées uniquement pour les appels de fonction synchrones.  Vous pouvez inclure des données sur l'interaction entre les couches dans toutes les méthodes de profilage.
