# Comment : cr&#233;er manuellement des sessions de performance avec les outils de profilage
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Vous pouvez créer manuellement une session de performance.  Pour ce faire, vous n'êtes pas obligé d'ouvrir un projet dans [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)].  Pour plus d’informations, consultez [Configuration de sessions de performance](../profiling/configuring-performance-sessions.md).  
  
### Pour créer manuellement une session de performance  
  
1.  Dans le menu **Analyser**, pointez sur **Profileur**, puis cliquez sur **Nouvelle session de performance**.  
  
     Une session de performance vide est ajoutée à l'**Explorateur de performances**.  
  
2.  Cliquez avec le bouton droit sur **Cibles**, puis cliquez sur **Ajouter un fichier binaire cible**.  
  
3.  Dans la boîte de dialogue **Ajouter un fichier binaire cible**, sélectionnez le nom du fichier, puis cliquez sur **Ouvrir**.  
  
     Un nouveau binaire est ajouté.
