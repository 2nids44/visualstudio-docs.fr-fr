# Cœurs logiques de l&#39;affichage Cœurs
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Chaque nom de canal représente l'un des cœurs logiques qui existent sur le système profilé.  L'attribution de nom réalise le décompte de 0 jusqu'au nombre total de cœurs logiques.  Par exemple, un système quadricœurs réalise le décompte depuis le cœur logique à jusqu'au cœur logique 3.
