# Connecteur de thread
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Lorsque vous cliquez sur un segment bloquant pour consulter une pile d'appels et sa pile de déblocage, le thread que le connecteur de thread prêt peut également s'afficher.  Si l'événement de déblocage s'est produit sur un autre thread dans le processus actuel, le connecteur de thread prêt identifie visuellement le thread et le segment d'exécution qui ont permis au thread bloqué de reprendre son exécution.
