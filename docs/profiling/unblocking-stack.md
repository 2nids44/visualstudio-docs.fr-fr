# Pile de d&#233;blocage
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Si l'élément de thread actuellement sélectionné représente un segment bloqué qui a ensuite commencé à s'exécuter après avoir été débloqué par un autre thread dans le processus courant, la pile des appels pour le thread à l'origine du déblocage est indiquée sous cet onglet.
