# Contr&#244;le Zoom (vue Utilisation)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Le contrôle de zoom vous permet d'effectuer des zooms avant et arrière sur le graphique d'utilisation du CPU pour se concentrer sur les zones d'intérêt particulier.  Ce contrôle zoom avant sur le centre de la vue.  Par conséquent, il est conseillé de centrer la zone d'intérêt avant de faire le zoom.  
  
 Vous pouvez faire glisser le pointeur de la souris dans le graphique d'utilisation du CPU ou dans le graphique d'activité de GPU pour créer une zone en surbrillance.  Lorsque le bouton de la souris est relâché, la vue zoom avant sur la plage sélectionnée.
