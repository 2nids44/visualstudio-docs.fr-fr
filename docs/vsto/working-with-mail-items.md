# <a name="work-with-mail-items"></a>Travailler avec des éléments de messagerie
  Le <xref:Microsoft.Office.Interop.Outlook.MailItem> classe contient les méthodes et les propriétés utilisées pour effectuer des tâches avec des messages électroniques.  
  
|Tâche|Procédure|  
|----------|---------------|  
|Créer un message électronique.|[Comment : créer par programmation un élément de courrier électronique](../vsto/how-to-programmatically-create-an-e-mail-item.md)|  
|Envoie un message électronique.|[Comment : envoyer par programme un e-mail](../vsto/how-to-programmatically-send-e-mail-programmatically.md)|  
|Répondre aux messages électroniques avec une action personnalisée.|[Comment : effectuer des actions par programme lors de la réception d’un message électronique](../vsto/how-to-programmatically-perform-actions-when-an-e-mail-message-is-received.md)|  
|Enregistrer les pièces jointes.|[Comment : enregistrer des pièces jointes à partir d’éléments de messagerie Outlook par programmation](../vsto/how-to-programmatically-save-attachments-from-outlook-e-mail-items.md)|  
|Ajouter des pièces jointes|[Comment : joindre des fichiers aux éléments de messagerie Outlook par programmation](../vsto/how-to-programmatically-attach-files-to-outlook-e-mail-items.md)|  
|Parcourir tous les messages non lus.|[Comment : récupérer par programme des messages non lus à partir de la boîte de réception](../vsto/how-to-programmatically-retrieve-unread-messages-from-the-inbox.md)|  
  
 Pour plus d’informations sur les tâches et le modèle objet Outlook, consultez [vue d’ensemble du modèle d’objet Outlook](../vsto/outlook-object-model-overview.md).
