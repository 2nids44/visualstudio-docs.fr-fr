# R&#233;f&#233;rence des API non manag&#233;es (D&#233;veloppement Office dans Visual Studio)
  À compter de la version 2007 de Microsoft Office system, les applications Office utilisent l’interface [Interface IManagedAddin](../vsto/imanagedaddin-interface.md) pour passer des appels dans un composant de chargeur de complément VSTO fourni avec [!INCLUDE[vsto_runtime](../vsto/includes/vsto-runtime-md.md)]. Ce composant vise à faciliter le chargement des compléments VSTO managés. Vous pouvez créer votre propre composant de chargeur de complément VSTO en implémentant cette interface.  
  
## Dans cette section  
 [Interface IManagedAddin](../vsto/imanagedaddin-interface.md)  
 Interface COM que vous pouvez implémenter pour charger et décharger les compléments VSTO managés dans les applications Office.
